## ✨ 核心能力

Graphic Design Pro 是一个基于 AI 的专业平面设计与 VI（视觉识别系统）生成技能，融合多模型生图能力与设计方法论，提供从需求到落地的完整工作流。

### 🎯 支持的设计类型

| 类别 | 涵盖内容 |
|------|---------|
| **品牌类** | 品牌设计、VI、CI、品牌视觉、品牌升级、品牌重塑、企业形象、品牌手册、品牌规范 |
| **Logo设计** | Logo、标志、商标、图标、Icon、动态标识、徽标 |
| **物料设计** | 海报、名片、招牌、易拉宝、宣传册、折页、画册、传单、DM、菜单 |
| **包装设计** | 包装盒、刀图、食品标签、礼盒、瓶贴、化妆品包装 |
| **户外广告** | 门头、店面招牌、灯箱、展架、展台、展览设计 |
| **电商设计** | 主图、详情页、Banner、店铺装修、社交媒体图、公众号配图、小红书图 |
| **VI系统** | 辅助图形、色彩系统、字体系统、排版规范、设计系统、品牌物料、衍生品 |

- **canvas-design** — 设计哲学方法论 + 81 字体 + 画布排版

## 📋 工作流

### 6 阶段完整流程

```
Phase 1: 需求追问 (Clarify)
    ↓
Phase 2: 竞品分析 + 设计哲学创建
    ↓
Phase 3: Moodboard + 样稿生成
    ↓
Phase 4: VI 物料扩展
    ↓
Phase 5: 合规审查 + 审美自检
    ↓
Phase 6: 落地指导 + 交付
```

### 模式路由（智能匹配项目复杂度）

| 模式 | 适用场景 |
|------|---------|
| 🟢 轻量级 | 单一物料（一张海报/一个Logo） |
| 🟡 中等 | 小型品牌套件（Logo+名片+1-2物料） |
| 🔵 深度 | 完整VI系统（多物料+多场景） |
| 🔴 重构 | 已有品牌升级/重塑 |

然后描述你的设计需求，例如：
- "帮我设计一个咖啡店的品牌VI"
- "做一个科技公司的Logo"
- "设计一套外卖包装"

## 📌 核心原则

1. **先追问，后生成** — 不急于出图，先确保需求清晰
2. **品牌一致性铁律** — 确认后全流程锁死
3. **合规内置** — 生成时即扫描，不是事后补审
4. **风格统一** — 所有物料共享同一视觉基因
5. **输出实用** — 矢量/PSD 可编辑

# Phase 2: 竞品分析 + 设计哲学创建 + 品牌战略定义

> 本文件为 Phase 2 的完整指令。由 SKILL.md 路由按需加载，不默认注入上下文。

**⚠️ 进入 Phase 2 前，必须先读取 `brand-profile.md` 确认品牌名锁定项。**

需求确认后，先做竞品视觉分析，再创建设计哲学（.md文件），作为后续所有生图和视觉创作的统一指导。

融合 canvas-design 方法论 + brand-cog 品牌战略框架 + 竞品视觉分析：为每套设计创建一个**视觉哲学/美学运动**作为创作纲领，同时定义**品牌战略系统**确保视觉与定位一致。

## 2.1 竞品视觉分析（品牌类设计必做）

对于品牌类设计（整套VI、品牌套件、Logo+品牌系统），在创建设计哲学之前，先做竞品视觉分析：

1. 读取 `references/vi-design-toolkit.md` §3 获取竞品分析模板
2. 根据用户提供的竞品（或自行搜索行业头部品牌），逐一分析：
   - Logo风格（字标/图标/组合？抽象/具象？）
   - 主色系（色相/饱和度/明度？冷色/暖色？）
   - 字体选择（衬线/无衬线？粗/细？）
   - 视觉风格（极简/装饰/科技/自然/奢华/活泼？）
   - 辅助图形（有无？类型？）
3. 总结行业视觉趋势，提出**差异化方向建议**

## 2.2 Moodboard制作（推荐）

在创建设计哲学前，建议先制作Moodboard收集视觉方向参考：

1. 读取 `references/vi-design-toolkit.md` §4 获取Moodboard制作指南
2. 从品牌定位推导3-5个视觉关键词
3. 按色彩/字体/构图/纹理/摄影/调性6个维度收集参考
4. 使用 color-palette 从参考图提取色彩
5. 与用户确认Moodboard传达的感觉是否正确

## 品牌战略框架（brand-cog 方法论）

对于品牌类设计（整套VI、品牌套件、Logo+品牌系统），在创建设计哲学之前，先用以下框架定义品牌战略：

| 维度 | 内容 | 示例 |
|------|------|------|
| **品牌个性** | 3-5个关键词 | "温暖、专业、不居高临下、现代" |
| **目标受众** | 年龄、职业、心理画像 | "22-35岁压力职场人，首次探索心理健康工具" |
| **竞争语境** | 核心竞品+差异化方向 | "vs Calm/Headspace：少冥想，多实用" |
| **品牌声音** | 文字风格和语气 | "知识型朋友，不宣教、不说教" |
| **视觉个性映射** | 个性→视觉特征翻译 | 见下方映射表 |

**品牌个性→视觉特征映射表**（来自 brand-cog）：

| 品牌个性 | 视觉特征 | 色彩倾向 | 字体倾向 |
|----------|---------|---------|---------|
| **奢华** | 极简、优雅、精致 | 金色、黑色、深沉色调 | 衬线体、细字重 |
| **活泼** | 大胆、动感、活力 | 鲜艳、饱和 | 圆角无衬线 |
| **专业** | 干净、结构化、可信赖 | 蓝色、灰色、白色 | 经典无衬线 |
| **自然/生态** | 有机、大地、温暖 | 绿色、棕色、奶油色 | 人文主义字体 |
| **科技/现代** | 几何、未来、极简 | 电蓝、暗色模式 | 几何无衬线 |
| **友好** | 柔和、亲和、温暖 | 粉彩、暖色调 | 圆角、友好字体 |
| **国潮** | 东方韵味、传统新生 | 朱红、墨黑、鎏金 | 书法体+无衬线混排 |
| **极简** | 留白、克制、呼吸感 | 单色/双色、大量留白 | 细线体、超轻字重 |

## 设计哲学创建方法

1. **命名运动**（1-2个词）: 如 "Brutalist Joy" / "Chromatic Silence" / "东方墨韵" / "数字禅意"
2. **阐述哲学**（4-6段，简明但完整）, 通过以下方面捕捉视觉本质:
   - 空间与形式 (Space and form)
   - 色彩与材质 (Color and material)
   - 比例与节奏 (Scale and rhythm)
   - 构图与平衡 (Composition and balance)
   - 视觉层级 (Visual hierarchy)

> **审美锚点**：设计哲学的创建和验证应参照 `references/design-aesthetics.md`，确保格式塔原则、Itten色彩理论、网格系统等被正确运用。

## 哲学示例

| 运动名称 | 哲学核心 | 视觉表达特征 |
|----------|---------|-------------|
| **Concrete Poetry** | 通过纪念碑式形态和大胆几何进行沟通 | 巨大色块、雕塑式排版、粗野主义空间分割 |
| **Chromatic Language** | 色彩作为主要信息系统 | 几何精确度，色彩区域创造意义，排版极简 |
| **Analog Meditation** | 通过纹理和呼吸空间进行安静的视觉冥想 | 纸张纹理、墨水渗透、巨大负空间 |
| **东方墨韵** | 水墨留白的东方哲思 | 大面积负空间、墨色渐变、纤细衬线字体 |
| **Digital Zen** | 科技与禅意的碰撞 | 精密网格、冷色调、代码美学×极简主义 |

## 关键指南

| 原则 | 说明 |
|------|------|
| **避免冗余** | 每个设计方面只提及一次，除非增加新深度 |
| **反复强调工艺** | 必须多次强调最终作品应看起来经过无数小时精心制作 |
| **留创作空间** | 保持具体的美学方向，但简洁到让AI有极高工艺水准的诠释选择空间 |
| **视觉优先** | 想法通过空间、形式、色彩、构图传达，而非段落 |
| **极简文字** | 文字始终是稀疏的、仅必要的，作为视觉元素整合 |
| **纯设计** | 制作艺术物件(ART OBJECTS)，而非带装饰的文档 |

## 微妙参考原则

将品牌/行业主题以**微妙、小众的参考**嵌入设计中——不总是字面的，总是精致的：
- 熟悉主题的人应直觉地感受到
- 其他人则只是体验一幅精湛的视觉构图
- 类比：像爵士音乐家引用另一首歌——只有知者能捕捉，但人人都欣赏音乐

## 核心理念

- 将用户的微妙输入作为基础，但**不约束创作自由**
- 创建一个美学运动宣言，下一阶段视觉化表达时遵循此哲学
- 设计哲学 = 90%视觉方向指引 + 10%必要文字约束

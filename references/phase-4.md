# Phase 4: VI物料扩展

> 本文件为 Phase 4 的完整指令。由 SKILL.md 路由按需加载，不默认注入上下文。

**⚠️ 进入 Phase 4 前，必须先读取 `brand-profile.md` 确认所有锁定项，并逐物料过一致性自检清单。**

> **⚠️ 合规前置（R9）**：在生成每个物料前，先扫描 `references/design-compliance.md` 中与该物料类型相关的合规要求，将关键约束内化到 prompt/设计参数中。特别关注：
> - 招牌/广告：中文字号≥英文字号、中文位置突出、商场额外要求
> - 包装：刀图/展开图（R10）、声称"有机"需认证、食品强制标签区域（GB 7718+GB 28050）、CMYK+Pantone色值标注
> - 所有物料：不得单独使用外文、绝对化用语黑名单、印刷出血+安全区

> **⚠️ 包装物料双产出规则（R10）**：涉及包装盒/手提袋/标签等实体包装时，必须同时提供：
> ① **结构刀图/展开图**（`.svg`格式，含刀版线、尺寸标注、图文位置）——供生产印刷用
> ② **成品效果图/Mockup**（`.svg`/`.png`格式，3D透视或精确渲染图）——供设计确认和展示用
> 两者缺一不可。

用户确认样稿方向后，根据设计类型扩展为全套VI物料。**所有物料共享同一设计哲学和品牌战略**。

## 品牌套件组件（brand-cog 方法论）

完整的品牌套件应包含以下组件（根据项目规模选取）：

| 组件 | 说明 | 优先级 |
|------|------|--------|
| **主Logo** | 主要使用场景的Logo | 🔴 必须 |
| **Logo变体** | 图标版/纯字版/竖版/横版/暗色版 | 🔴 必须 |
| **色彩系统** | 主色/辅色/强调色/中性色（含HEX值） | 🔴 必须 |
| **字体系统** | 标题字体/正文字体/层级/字重 | 🔴 必须 |
| **图片风格** | 摄影风格、插画指南 | 🟡 推荐 |
| **品牌声音** | 品牌书面语感和个性 | 🟡 推荐 |
| **社交媒体模板** | 头像/封面/帖子模板 | 🟡 推荐 |
| **商务物料** | 名片/信纸/邮件签名 | 🟡 推荐 |
| **品牌规范手册** | 视觉识别使用规范文档 | 🟢 可选 |

**执行原则**：
- 🔴 必须组件：每个品牌项目都要产出
- 🟡 推荐组件：根据行业和预算推荐，用户确认后产出
- 🟢 可选组件：大型项目或特殊需求时产出

## VI物料清单

读取 `references/vi-materials.md` 获取完整物料清单和VI规范刚性要求。核心分类：

| 类别 | 典型物料 |
|------|---------|
| 基础系统 | Logo规范、标准色、标准字、组合规范 |
| 办公系统 | 名片、信纸、信封、文件夹、工牌 |
| 导视系统 | 门牌、指示牌、楼层导视 |
| 宣传系统 | 海报、折页、画册、易拉宝 |
| 戶外系统 | 招牌、灯箱、道旗、车身广告 |
| 包装系统 | 手提袋、包装盒、标签 |
| 数字系统 | 微信头图、PPT模板、邮件签名 |
| **Logo规范** | **最小尺寸、安全空间、禁用示例、响应式变体** |

**VI刚性规范**（`references/vi-materials.md` H节）：
- Logo最小尺寸：印刷≥15mm，手机≥36px
- Logo安全空间：四周留白≥Logo高度1/2
- Logo禁用10种错误用法：拉伸/旋转/投影/描边/换色/复杂背景/入侵安全空间/小于最小尺寸/裁切/叠加文字
- 响应式Logo：完整版→标准版→紧凑版→图标版→反白版→单色版
- 色彩系统标注：RGB + HEX + CMYK + Pantone 四种色值同时标注
- 字体系统层级：H1-H3 + Body + Caption 五级

## 执行策略

1. **先与用户确认物料清单**：根据行业和场景推荐物料，用户增减确认
2. **色彩系统先行**：使用 `scripts/color_system.py` 生成品牌色彩系统（主色→辅色→强调色→中性色→色阶→渐变→WCAG检测）
3. **Logo矢量绘制**：使用 svg-draw 技能直接编写SVG代码，保证Logo精确可控
4. **逐物料生成**：每类物料生成1-2稿，**用户确认后继续**（R8铁律）
5. **风格统一**：所有物料共享同一设计哲学和色彩/字体/构图规则
6. **尺寸适配**：每种物料使用对应的正确尺寸
7. **每个物料生成时即时预扫描合规**（Phase 5的快速版）

## 多物料排版策略

- AI生图物料（招牌/海报/易拉宝）：用豆包即梦生成主视觉
- 精细排版物料（名片/信纸/画册）：用 canvas-design 画布方法精确排版
- 两种方式产出风格统一、品质一致的成品

## 色彩系统生成（color_system.py）

VI设计的色彩系统应使用脚本自动生成，确保色阶完整和WCAG合规：

```bash
# 从主色自动生成完整色彩系统
python scripts/color_system.py "#1A73E8" --output brand-colors.json

# 手动指定辅色和强调色
python scripts/color_system.py "#1A73E8" --secondary "#E74C3C" --accent "#F59E0B" --output brand-colors.json
```

**自动生成内容**：
- 主色9级色阶（50-900）
- 辅色（默认互补色）9级色阶
- 强调色（默认类似色）9级色阶
- 中性色7级色阶
- 3组渐变方案
- WCAG无障碍对比度检测

**色彩提取**（color-palette 集成）：
- 如用户已有参考图片，使用 `~/.workbuddy/skills/color-palette/scripts/extract_palette.py` 提取主色
- 提取后作为 `color_system.py` 的输入，生成完整色阶

**色彩使用比例**（读取 `references/vi-design-toolkit.md` §5）：
- 主色60% + 辅助色30% + 强调色10%（60-30-10法则）
- 各场景比例适配：海报50-60/25-35/10-15，名片40-50/30-40/10-20，网站60-70/20-30/5-10

## 品牌规范手册生成（brand_guidelines.py）

Phase 4 完成后，自动生成品牌视觉识别规范手册：

```bash
# 从配置文件生成
python scripts/brand_guidelines.py "品牌名" --config brand-config.json --output brand-guidelines.md
```

**手册包含8大模块**：品牌概述 → Logo规范 → 色彩系统 → 字体系统 → 图形元素 → 排版规范 → 应用场景 → 禁用规范

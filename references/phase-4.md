# Phase 4: VI物料扩展

> 本文件为 Phase 4 的完整指令。由 SKILL.md 路由按需加载，不默认注入上下文。
> **执行原则：按 Layer 递进，过 Gate 解锁。禁止一次性生成全部物料。**

---

## P4-G4 · 进入门（强制执行）

**进入 Phase 4 前，必须：**
1. 读取 `brand-profile.md` 确认所有锁定项
2. 确认 Phase 3 已通过 P4-G3
3. 逐物料过一致性自检清单（R1-R11）

**未过 P4-G4 禁止开始 Phase 4。**

---

## Layer 1 · 表面层（物料清单确认 + 色彩系统）

> **目标：确认「做什么物料」，不开始做任何设计。**
> **禁止在 Layer 1 生成任何物料。**

### 此刻只做 2 件事：

1. **推荐物料清单**（根据行业和场景，给出推荐列表）
2. **生成色彩系统**（使用 `scripts/color_system.py`）

### 物料清单推荐格式

```
推荐物料清单
━━━━━━━━━━━━━━━━━━━
🔴 必须（建议全部保留）：
• {物料A} · {说明}
• {物料B} · {说明}

🟡 推荐（根据预算选择）：
• {物料C} · {说明}
• {物料D} · {说明}

🟢 可选（大型项目再考虑）：
• {物料E} · {说明}
```

### 色彩系统生成

```bash
python scripts/color_system.py "{主色HEX}" --output brand-colors.json
```

**只生成色彩系统文件，不开始设计物料。**

### Layer 1 执行规则

- **不给用户 overwhelm**：推荐 ≤8 个物料（必须+推荐）
- 用户可增减，确认后再开始设计
- 包装类物料在清单中标注「含刀图+效果图（R10）」

### P4-G1 · 方向确认门

```
📍 Phase 4 · Layer 1 完成
━━━━━━━━━━━━━━━━━━━
推荐物料：{N}项
色彩系统：已生成（{brand-colors.json 路径}）

请确认：
1. 物料清单是否需要调整？（保留 / 增减）
2. 色彩系统是否符合预期？（是 / 需调整）
3. 是否开始逐个设计物料？（是 / 否）
```

**用户未确认清单 → 禁止进入 Layer 2。**

## Layer 2 · 细节层（逐物料生成 + 逐个确认）

> **目标：基于确认的清单，逐个设计物料，每做一个确认一个。**
> **仅在用户通过 P4-G1 后解锁。**

### 2.1 执行策略

**逐个物料生成，不是批量生成。**

```
物料 1/8：名片
→ 生成 1-2 稿
→ 用户确认
→ 通过 → 下一个物料
→ 不通过 → 修改后再确认

物料 2/8：信纸
→ ...
```

### 2.2 每个物料的生成流程

**每个物料开工前必过「位置四问」**（融入huashu-design）：见 SKILL.md §位置四问。

1. **合规前置扫描（R9）**：生成前扫描 `references/design-compliance.md` 中该物料的关键约束（只扫描最相关的 3-5 项，不是全部 20 项）
2. **构造 Prompt**：基于 brand-profile.md + 设计哲学 + 预渲染素材（R11）
3. **素材质量门（5-10-2-8）**：如需参考图/产品图/背景素材，执行 5-10-2-8 门槛。见 SKILL.md §5-10-2-8。不够 8 分宁可不用，改用 honest placeholder。
4. **生成 1-2 稿**
4. **一致性自检**：过 R1-R11 清单（只过与当前物料相关的项）
5. **一致性自检**：过 R1-R11 清单（只过与当前物料相关的项）
6. **展示并等待确认**

### 2.3 多物料排版策略

| 物料类型 | 生成方式 |
|---------|---------|
| 招牌/海报/易拉宝 | 豆包即梦生图 |
| 名片/信纸/画册 | canvas-design 精确排版 |
| 包装 | 生图效果图 + SVG 刀图（R10双产出） |

### 2.4 品牌规范手册框架

Phase 4 进行过程中，同步填充品牌规范手册框架（Markdown 版），为 Phase 6 的 PPT 做准备。

### Layer 2 执行规则

- **每次只做一个物料**，用户确认后再做下一个
- 使用预渲染素材（R11）：Logo/品牌名/辅助图形直接引用 `assets/` 中的 PNG，不重新生成
- 包装类物料必须同时提供刀图+效果图（R10）
- 每个物料生成时标注 CMYK + Pantone（印刷物料）

### P4-G2 · 细节确认门（每个物料后）

```
📍 Phase 4 · Layer 2 · 物料 {N}/{总}
━━━━━━━━━━━━━━━━━━━
物料：{物料名}

请确认：
1. 本物料是否满意？（满意 / 需修改）
2. 是否继续下一个物料（{下一个物料名}）？（是 / 否，先停在这里）
```

**所有物料完成后，进入 P4-G3：**

```
📍 Phase 4 · Layer 2 · 全部物料完成
━━━━━━━━━━━━━━━━━━━
已完成物料：{列表}

请确认：
1. 全部物料是否满意？（满意 / 某个需修改）
2. 是否进入高级定制（响应式Logo/完整规范手册/特殊工艺）？（是 / 否，直接审查）
```

## Layer 3 · 高级层（响应式 Logo + 完整规范手册 + 特殊工艺）

> **目标：为大型项目提供完整品牌规范。**
> **仅在用户通过 P4-G2 且主动要求/场景需要时解锁。**

### 响应式 Logo 系统

- 完整版 → 标准版 → 紧凑版 → 图标版 → 反白版 → 单色版

### 完整品牌规范手册

使用 `scripts/brand_guidelines.py` 生成完整手册（8大模块）。

### 特殊工艺标注

- 烫金/UV/压凹/模切等工艺标注
- 供应商建议（如用户需要）

### P4-G3 · Phase 验收门

```
📍 Phase 4 完成 · 最终确认
━━━━━━━━━━━━━━━━━━━
全部物料已产出并确认：{列表}

请确认：
1. 全部物料一致性自检通过？（是 / 否，某物料需回退）
2. 是否进入 Phase 5（合规审查+审美自检）？（是 / 否）
```

## 关键提示

- **R8 铁律**：每个物料逐个生成并经用户确认。禁止批量做完再展示。
- **R9 合规前置**：每个物料生成前即时预扫描合规，不是事后补审。
- **R10 包装双产出**：包装必须同时提供刀图+效果图，缺一不可。
- **R11 素材复用**：跨物料重复元素使用预渲染素材，不得重绘。
- 色彩系统使用比例：主色60% + 辅助色30% + 强调色10%。

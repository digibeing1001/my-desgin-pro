## 1. 语言文字规范

### 1.1 法律依据

- 《中华人民共和国国家通用语言文字法》（2001年施行，2025年修订）
- 各省/市实施办法（如《上海市实施〈国家通用语言文字法〉办法》）

### 1.2 核心要求

| 要求 | 法条 | 说明 |
|------|------|------|
| **规范汉字为基本用字** | §13 | 公共服务行业以规范汉字为基本服务用字 |
| **中文为主、外文为辅** | §14 | 招牌、广告、告示、标志牌等使用外国文字并同时使用中文的，应当使用规范汉字，**并以规范汉字为主** |
| **不得单独使用外文** | §14 | 如需使用外国文字，应当与规范汉字同时使用（注册商标、国际通用标志除外） |
| **不得单独使用拼音** | §14 | 不得单独使用汉语拼音（注册商标除外），需与规范汉字同时使用 |
| **公共场所用字规范** | §15 | 公共场所设施用字、招牌广告用字应以国家通用语言文字为基本用字 |
| **禁止繁体/异体/自造字** | §14 | 招牌、广告用字应使用规范汉字（姓氏、书法艺术、历史文物等除外） |

### 1.3 设计实操要点

- **中文字号 ≥ 英文字号**：这是"中文为主"的核心体现，各地招牌规范均有此要求
- **中文位置突出**：中文应处于视觉主导位置（上方/左侧/中央），英文/外文为辅助
- **同时出现原则**：凡有外文处，必须有对应中文，不得仅有外文
- **字体选择**：使用规范简化字字体（如思源黑体、方正系列等），避免使用含异体字的书法字体
- **翻译一致**：外文翻译应与中文含义相符，不得出现翻译错误或歧义

## 2. 商标使用规范

### 2.1 法律依据

- 《中华人民共和国商标法》（2019年修订）
- 《商标法实施条例》

### 2.2 核心要求

| 要求 | 法条 | 说明 |
|------|------|------|
| **注册商标标®** | §63 | 经商标局核准注册的商标为注册商标，可标注®或"注册商标"字样 |
| **未注册禁标®** | §63 | 未注册商标不得标注®，否则构成冒充注册商标违法行为 |
| **商标规范使用** | §49 | 注册商标不得自行改变注册事项（文字/图形/颜色等），否则限期改正 |
| **不得侵权** | §57 | 未经许可不得在同种/类似商品上使用相同/近似商标 |
| **不得淡化** | §58 | 不得将他人商标用作通用名称 |
| **禁止不良影响** | §10 | 有害于社会主义道德风尚或有其他不良影响的标志不得作为商标 |

### 2.3 商标法§10禁用标志（设计绝对禁区）

| 禁用类型 | 说明 |
|----------|------|
| 国家名称/国旗/国徽/国歌/军旗/军徽/军歌/勋章 | 中国及外国的均不可 |
| 中央国家机关名称/标志/所在地特定名称 | 如"中南海""人民大会堂" |
| 国际组织名称/旗帜/徽记 | 除非该组织同意 |
| 官方标志/检验印记 | 除非授权 |
| 红十字/红新月标志 | 国际人道主义标志 |
| 带有民族歧视性 | 任何民族歧视元素 |
| 带有欺骗性 | 容易使公众对商品质量/产地产生误认 |
| 有害于社会主义道德风尚/其他不良影响 | 含低俗/暴力/色暗含义 |

### 2.4 设计实操要点

- 客户提供注册商标 → 标注®，确认注册号
- 客户商标未注册 → 不得标注®，可标注™
- 设计中使用的字体/图形不得侵犯他人商标权
- Logo设计中避免使用国旗、国徽、国际组织标志等（§10禁用标志）
- 避免使用县级以上行政区划地名作为商标（§10）
- **Logo设计前做近似检索**：在中国商标网查询同类别的相似商标

## 3. 广告内容规范

### 3.1 法律依据

- 《中华人民共和国广告法》（2021年修订）
- 《广告语言文字管理暂行规定》（国家工商总局）

### 3.2 核心要求

| 要求 | 法条 | 说明 |
|------|------|------|
| **禁止绝对化用语** | §9(3) | 不得使用"国家级""最高级""最佳""第一""顶级""极品"等 |
| **禁止虚假宣传** | §4 | 广告不得含有虚假或引人误解的内容，不得欺骗/误导消费者 |
| **禁止贬低他人** | §13 | 不得贬低其他生产经营者的商品或服务 |
| **数据需标来源** | §11 | 使用数据/统计/调查结果等应真实准确并标明出处 |
| **引用语需真实** | §11 | 引用他人名义/言论的应真实并有书面证明 |
| **承诺需可兑现** | §9(2) | 对性能/功能的表示应当准确/清楚/明白 |

### 3.3 绝对化用语黑名单（高频踩雷）

| 禁用词 | 替代建议 |
|--------|---------|
| 最/最佳/最好 | 优选/精选/品质之选 |
| 第一/No.1 | 领先/前列 |
| 顶级/极品 | 高端/卓越 |
| 独一无二/绝无仅有 | 独特/专属 |
| 全网最低 | 优惠/特价 |
| 100%/零风险 | 显著提升/低风险 |

### 3.4 设计实操要点

- 文案中出现"最""第一"等字眼时立即警告
- 数据/奖项引用需标注来源和年份
- "前后对比"式广告需有真实数据支撑
- 价格标示需明确是否含税、有效期等

## 4. 字体版权合规规范

### 4.1 法律依据

- 《中华人民共和国著作权法》（2021年修订）
- 《计算机软件保护条例》
- 各字体厂商《最终用户许可协议》(EULA)

### 4.2 字体版权核心规则

| 规则 | 说明 | 风险等级 |
|------|------|---------|
| **商业使用需授权** | 绝大多数字体需购买商用授权，个人免费不等于商用免费 | 致命 |
| **嵌入式授权** | 将字体嵌入PDF/网页需额外"嵌入式授权" | 致命 |
| **改编权限制** | 对字体进行变形/描边/渐变等修改可能侵犯改编权 | 高 |
| **转发/分发限制** | 将字体文件随设计文件发送给客户需确认EULA | 高 |

### 4.3 常见侵权场景

| 场景 | 风险 | 正确做法 |
|------|------|---------|
| 设计稿中使用未授权字体 | 客户落地使用时被字体商起诉 | 仅使用已授权字体 |
| 将字体文件随设计交付 | 侵犯字体厂商的分发权 | 交付PDF/图片，不附字体文件 |
| 网页@font-face嵌入 | 需要Web授权许可 | 使用Web Font服务或开源字体 |
| Logo中使用付费字体轮廓 | Logo需转化为路径/图形 | 确认字体EULA允许转为路径 |

### 4.4 安全字体清单

**免费商用中文字体**（无需授权，可放心使用）：

| 字体 | 类型 | 适用场景 |
|------|------|---------|
| 思源黑体 (Noto Sans CJK) | 无衬线 | 通用正文/标题 |
| 思源宋体 (Noto Serif CJK) | 衬线 | 文化/传统/正文 |
| 站酷快乐体 | 手写风 | 活泼/儿童/休闲 |
| 站酷小薇LOGO体 | 装饰 | Logo/标题 |
| 阿里巴巴普惠体 | 无衬线 | 电商/互联网 |
| OPPO Sans | 无衬线 | 科技/现代 |
| 江城正义体 | 无衬线 | 通用 |
| 文泉驿微米黑 | 无衬线 | 通用 |
| 方正黑体/方正书宋 | 无衬线/衬线 | 方正4款免费：黑体/书宋/仿宋/楷体 |

**高风险字体（必须购买商用授权）**：
- 方正系（除4款免费外的所有字体） — 方正维权极为积极，常见5-10万赔偿案例
- 汉仪系 — 维权活跃
- 华康系 — 维权活跃
- Monotype/Helvetica — 国际字体，授权费用高
- Adobe Fonts — 需Creative Cloud订阅，仅限个人使用

### 4.5 设计实操要点

- 每次使用字体前，先确认其商用授权状态
- canvas-design 技能内置81个OFL开源字体，优先使用这些字体
- 客户指定字体时，要求客户提供授权证明
- AI生图中的文字应被视为占位符，落地时替换为授权字体

## 5. 著作权与AI生成物规范

### 5.1 法律依据

- 《中华人民共和国著作权法》（2021年修订）
- 《生成式人工智能服务管理暂行办法》（2023年）
- 北京互联网法院"AI文生图第一案"（2023）判决要点

### 5.2 AI生成物版权现状

| 问题 | 现状 | 应对策略 |
|------|------|---------|
| **AI生成图片是否有版权** | 中国法院认定：有人的智力投入（提示词设计、参数调整、筛选）的AI生成物可受著作权保护 | 保留创作过程记录（prompt版本、参数、筛选理由） |
| **AI训练数据侵权** | 争议中，目前无明确裁判规则 | 避免在prompt中直接引用特定艺术家风格名称 |
| **AI生成物近似他人作品** | 可能构成侵权 | 不使用"模仿XX风格""像XX作品"类prompt |
| **AI生成物的著作权归属** | 操作者（用户）享有著作权（需证明智力投入） | 合同中明确AI辅助设计成果的版权归属 |

### 5.3 设计实操要点

- **不引用在世艺术家姓名**：prompt中避免出现"像莫奈风格""类似草间弥生"等表述
- **保留创作证据**：记录每次生图的prompt、参数和选择理由，作为智力投入的证明
- **交付合同条款**：建议在合同中明确"本设计由AI辅助生成，著作权归甲方所有"
- **避免过度相似**：对生成结果做逆向图片搜索，确认与已有作品无实质相似
- **素材来源声明**：使用AI生成物时，向客户声明"设计素材由AI辅助生成"

## 6. 户外广告与招牌规范

### 6.1 法律依据

- 《城市户外广告和招牌设施技术标准》CJJ/T 149-2021
- 各城市地方规范（上海/北京/广州/深圳等）
- 《中华人民共和国城乡规划法》

### 6.2 招牌尺寸规范

| 参数 | 要求 | 来源 |
|------|------|------|
| **高度** | ≤ 店面层高的1/3 | CJJ/T 149-2021 |
| **牌匾宽度** | ≤ 3m（牌匾式） | 上海规范 |
| **箱体宽度** | 不超过店面宽度 | 各地规范 |
| **伸出距离** | ≤ 0.5m（平行式），≤ 1.5m（骑楼） | CJJ/T 149-2021 |
| **底部净高** | ≥ 2.5m（人行道上方） | CJJ/T 149-2021 |

### 6.3 招牌内容规范

| 要求 | 说明 |
|------|------|
| **店名/商号为主** | 招牌内容应为市场主体名称或规范化简称 |
| **中文为主** | 规范汉字处于版面主导地位，外文为辅 |
| **中文字号≥外文** | 中文字体不小于外文字体（各地规范通用要求） |
| **禁止单独外文** | 不应单独使用外国文字（注册商标除外） |
| **禁止拼音替代** | 不得以拼音代替汉字 |
| **内容与经营相符** | 招牌内容应与工商登记一致 |

### 6.4 设计实操要点

- 设计前确认当地城管/审批要求
- 招牌预留安装余量（出血3-5mm）
- 夜间照明方案需考虑光污染限制
- 避免使用反光材料影响交通安全
- 结构安全需专业审核（大型招牌）

### 6.5 商场/购物中心特殊规范

> 商场内租户招牌除遵守通用招牌规范外，还需满足商场统一的视觉管理要求。

**商场常见招牌要求**：

| 要求 | 说明 | 来源 |
|------|------|------|
| **中文字号≥英文** | 商场强制要求，且执行比城管更严格 | 各大商场装修管理手册 |
| **招牌尺寸统一** | 商场规定招牌区域尺寸，租户不得超出 | 商场租约条款 |
| **材质限制** | 部分商场禁用霓虹灯/喷绘，只允许发光字/不锈钢/亚克力 | 商场装修标准 |
| **背光/灯光规范** | 色温、亮度、是否允许动态灯光均有限制 | 商场装修标准 |
| **审批流程** | 招牌设计需经商场物业/企划部审批后方可制作安装 | 商场租约条款 |
| ** Logo占比** | 部分商场要求Logo+品牌名占比不超过招牌面积60% | 商场装修标准 |
| **安装方式** | 商场统一规定安装方式和结构要求 | 商场装修标准 |

**设计实操**：
- 做商场招牌前，先向客户索要商场《装修管理手册》和《视觉规范》
- 商场内招牌中文字号必须明显大于英文，不可打擦边球
- 提交效果图时需标注材质、灯光方式、安装节点

## 7. 特殊行业规范

### 7.1 医疗/药品/医疗器械

| 要求 | 法规 |
|------|------|
| 须标注医疗广告审查证明文号 | 《医疗广告管理办法》 |
| 禁止保证治愈/隐含保证 | 《广告法》§16 |
| 禁止利用患者/医生形象作推荐 | 《广告法》§16 |
| 非药品不得宣传治疗功效 | 《药品管理法》 |

### 7.2 酒类

| 要求 | 法规 |
|------|------|
| 不得诱导/劝饮 | 《广告法》§23 |
| 不得出现饮酒动作 | 《广告法》§23 |
| 不得明示/暗示消除紧张焦虑 | 《广告法》§23 |
| 不得针对未成年人 | 《广告法》§23 |

### 7.3 烟草

| 要求 | 法规 |
|------|------|
| 禁止在大众媒体发布广告 | 《广告法》§22 |
| 禁止变相发布 | 《广告法》§22 |

### 7.4 教育/培训

| 要求 | 法规 |
|------|------|
| 不得保证升学/通过考试 | 《广告法》§24 |
| 不得暗示有考试命题人参与 | 《广告法》§24 |
| 不得利用科研/学术机构名义 | 《广告法》§24 |

### 7.5 房地产

| 要求 | 法规 |
|------|------|
| 须标明建筑面积/套内面积 | 《广告法》§26 |
| 不得以到达时间表示距离 | 《广告法》§26 |
| 不得承诺升值/投资回报 | 《广告法》§26 |
| 须标明预售许可证号 | 《房地产广告发布规定》 |

### 7.6 食品/保健食品

| 要求 | 法规 |
|------|------|
| 不得宣传治疗功效 | 《食品安全法》 |
| 保健食品须标明"不能代替药物" | 《广告法》§18 |
| 不得使用医疗机构/医生名义 | 《广告法》§18 |
| **预包装食品营养标签** | 《GB 28050-2011 预包装食品营养标签通则》 |
| **食品接触材料标准** | 《GB 4806系列 食品安全国家标准 食品接触材料》 |
| **有机产品标识** | 《GB/T 19630-2019 有机产品 生产、加工、标识与管理体系要求》 |

**食品包装标签强制要素**（GB 7718 + GB 28050）：

| 要素 | 说明 | 位置要求 |
|------|------|---------|
| 食品名称 | 反映食品真实属性 | 主要展示版面 |
| 净含量/规格 | 按规定格式标注 | 主要展示版面 |
| 生产者信息 | 名称+地址+联系方式 | 任意版面 |
| 生产日期 | 年月日格式 | 清晰可辨位置 |
| 保质期 | "保质期至XXXX"或"保质期XX个月" | 与生产日期对应 |
| 贮存条件 | 如"冷藏0-4℃" | 与保质期相邻 |
| 配料表 | 按添加量降序排列 | 任意版面 |
| 营养成分表 | 能量+4核心营养素（蛋白质/脂肪/碳水/钠） | 任意版面 |
| 产品标准代号 | 执行标准号 | 任意版面 |
| 食品生产许可证编号 | SC+14位数字 | 任意版面 |

**有机认证标识规范**（GB/T 19630）：
- 未获得有机认证的产品，**不得**在包装上使用"有机""有机食品"等字样
- 已获认证的产品，必须标注：认证机构名称+有机认证标志+认证码
- 有机转换期产品标注"有机转换"而非"有机"
- 设计师在出包装稿时，如果品牌声称"有机"，必须提醒客户提供有机认证证书

## 9. 无障碍设计规范（WCAG）

### 9.1 法律与标准依据

- 《中华人民共和国残疾人保障法》
- 《无障碍环境建设法》（2023年施行）
- WCAG 2.1（Web Content Accessibility Guidelines）
- CJJ/T 149-2021 对户外标识的无障碍要求

### 9.2 色彩对比度要求

| 元素类型 | 最低对比度 | 推荐对比度 | WCAG等级 |
|----------|-----------|-----------|----------|
| 正文文字（< 18pt） | 4.5:1 | 7:1 | AA / AAA |
| 大文字（≥ 18pt 或 ≥ 14pt粗体） | 3:1 | 4.5:1 | AA / AAA |
| 图标/图形元素 | 3:1 | — | AA |
| 非活跃/装饰元素 | 无要求 | — | — |

### 9.3 色盲安全设计

| 要求 | 说明 |
|------|------|
| **不依赖单一色彩传达信息** | 色彩不能是传达信息的唯一手段，必须辅以文字/图标/纹理 |
| **红绿配色禁忌** | 红绿色盲占男性8%，避免红绿对比作为关键信息区分 |
| **验证工具** | 使用 Coblis 或 Color Oracle 模拟色盲视角 |
| **安全色组合** | 蓝-橙、蓝-黄、白-黑是最安全的配色组合 |

### 9.4 设计实操要点

- VI色彩系统必须包含WCAG对比度检测（`scripts/color_system.py`已实现）
- 每组前景色/背景色组合都需要验证对比度
- 户外标识还需考虑日光/弱光下的可读性
- 招牌字体大小需按视距计算（10米视距需≥150mm字号）

## 10. VI设计禁忌清单

> VI设计中最容易犯的错误，可能导致设计失败或法律风险。AI生图时尤其需要注意。

### 10.1 Logo设计禁忌

| 禁忌 | 说明 | 后果 |
|------|------|------|
| **Logo过于复杂** | 细节过多，小尺寸无法辨认 | 名片/工牌/社交媒体头像上识别困难 |
| **缺乏黑白版本** | 只有彩色版，无反白/墨稿 | 传真/报纸/单色印刷无法使用 |
| **无最小尺寸规范** | 没有规定Logo最小使用尺寸 | 缩小后模糊不清，损害品牌形象 |
| **无安全空间** | 没有规定Logo周围留白 | 其他元素挤压Logo，视觉混乱 |
| **使用渐变/投影** | Logo含渐变/阴影/3D效果 | 缩放失真、单色印刷困难、低成本工艺无法实现 |
| **字体未转路径** | Logo文字用活字而非路径 | 不同设备上显示不一致 |
| **模仿知名品牌** | 与知名品牌Logo相似 | 侵权诉讼、品牌混淆 |
| **方向性元素** | 箭头/人物朝右/朝外 | 暗示"离开"而非"吸引"，版面布局受限 |
| **过度装饰** | 加光效/描边/内阴影 | 降低专业性，限制使用场景 |

### 10.2 色彩设计禁忌

| 禁忌 | 说明 | 后果 |
|------|------|------|
| **色彩过多** | 使用超过4种主色 | 视觉混乱，无法建立品牌识别 |
| **忽视色彩心理学** | 儿童品牌用黑色，奢侈品用荧光色 | 与品牌定位严重脱节 |
| **未考虑文化差异** | 白色在东方=丧事，在西方=纯洁 | 国际品牌尤其注意 |
| **对比度不足** | 浅灰文字+白色背景 | 可读性差，违反WCAG |
| **仅靠色彩区分信息** | 图表/图标仅用颜色区分 | 色盲用户无法识别 |
| **忽视印刷色彩** | 设计用RGB，印刷转CMYK严重偏色 | 实际效果与设计稿差异巨大 |

### 10.3 排版设计禁忌

| 禁忌 | 说明 | 后果 |
|------|------|------|
| **字体超过3种** | 同一画面使用3种以上字体 | 视觉混乱，缺乏统一感 |
| **中英文排版不统一** | 中文用衬线，英文用无衬线，缺乏协调 | 视觉割裂感 |
| **行距过紧** | 行距小于字号的1.2倍 | 可读性极差 |
| **文字对齐不一致** | 同一页面混用左对齐/居中/右对齐 | 缺乏秩序感 |
| **孤行/寡行** | 段落首行/末行单独留在上一页/下一页 | 排版不专业 |

### 10.4 品牌套件禁忌

| 禁忌 | 说明 | 后果 |
|------|------|------|
| **VI设计与品牌定位脱节** | 高端品牌用卡通风格 | 品牌认知混乱 |
| **不同物料风格不统一** | 名片一个风格、海报另一个风格 | 无法建立统一品牌形象 |
| **只有Logo没有系统** | 仅设计Logo，无色彩/字体/图形规范 | 落地执行混乱，每家供应商做出来不一样 |
| **缺少禁用规范** | 未展示错误用法 | 员工/供应商错误使用，损害品牌 |
| **忽视应用场景** | 未考虑实际载体限制 | 设计无法落地或落地效果差 |
| **盲目跟风** | 直接复制当前流行风格 | 缺乏独特性，随时过时，可能侵权 |

## 11. 审查Checklist

每个物料生成后自动执行以下审查：

### A. 语言文字审查

- [ ] 中文是否为版面主导（字号≥外文、位置突出）
- [ ] 是否存在单独使用外文的情况（注册商标除外）
- [ ] 是否存在单独使用拼音的情况
- [ ] 是否使用了规范汉字（无繁体/异体/自造字，商标除外）
- [ ] 外文翻译是否与中文含义一致

### B. 商标审查

- [ ] 注册商标是否正确标注®
- [ ] 未注册商标是否误标®
- [ ] 设计元素是否可能侵犯他人在先权利
- [ ] Logo是否使用了商标法§10禁用标志

### C. 广告内容审查

- [ ] 是否包含绝对化用语（最/第一/顶级等）
- [ ] 数据/奖项引用是否标明来源
- [ ] 是否存在虚假/误导性内容
- [ ] 承诺性语言是否有依据

### D. 字体版权审查

- [ ] 所用字体是否具有商用授权
- [ ] 是否有字体文件随设计交付（不应有）
- [ ] Logo文字是否已转为路径
- [ ] AI生图文字是否标注为占位符

### E. 著作权审查

- [ ] Prompt是否引用了特定艺术家姓名
- [ ] 生成结果是否与已知作品过度相似
- [ ] 是否保留了创作过程记录
- [ ] 是否在合同中明确了版权归属

### F. 行业特殊审查

- [ ] 是否涉及特殊行业（医疗/药品/酒类/烟草/教育/房产/食品/金融/化妆品/互联网/零售/母婴/汽车/服装/文旅）
- [ ] 如涉及，是否满足该行业特殊广告要求
- [ ] 如涉及食品/化妆品/服装/零售，标签强制要素是否齐全
- [ ] 如涉及数码3C产品，3C认证/入网许可/能效标识是否标注
- [ ] 如涉及母婴儿童，安全警示和年龄标注是否到位
- [ ] 如涉及商场/购物中心内物料，是否符合商场视觉规范

### G. 印刷技术审查

- [ ] 色彩模式是否正确（印刷CMYK/屏幕RGB）
- [ ] 分辨率是否满足印刷要求
- [ ] 出血和安全区是否留足
- [ ] 黑色文字是否为单色黑

### H. 无障碍审查

- [ ] 前景色/背景色对比度是否≥ 4.5:1（正文）
- [ ] 是否仅靠色彩传达关键信息
- [ ] 红绿配色是否有替代区分手段
- [ ] 户外标识字号是否满足视距要求

### I. 户外招牌审查（如适用）

- [ ] 尺寸是否符合当地规范
- [ ] 中文字体是否≥外文字体
- [ ] 是否与工商登记一致
- [ ] 安装安全是否考虑

### J. VI设计禁忌审查

- [ ] Logo是否过于复杂（小尺寸可辨？）
- [ ] 是否有黑白/反白版本

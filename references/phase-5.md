> 本文件为 Phase 5 的完整指令。由 SKILL.md 路由按需加载，不默认注入上下文。

**进入 Phase 5 前，必须：**

## Layer 1 · 表面层（快速合规扫描 + 风险分级）

> **目标：快速识别「高危风险」，不展开全部审查。**
> **禁止在 Layer 1 输出完整审查报告。**

### 此刻只做 3 件事：

1. **高危项快速扫描**（只扫描最关键的 5 项）
2. **风险分级**（红/黄/绿）
3. **如有红旗，立即提出修正方案**

### 高危 5 项速查

| # | 审查项 | 违反后果 | 快速检查方法 |
|---|--------|---------|-------------|
| 1 | 商标法§10禁用标志 | 法律风险 | 是否含国旗/国徽/红十字等？ |
| 2 | 绝对化用语 | 行政处罚 | 是否含"最/第一/顶级"？ |
| 3 | 中文字体为主（招牌/广告） | 合规风险 | 中文是否≥英文？ |
| 4 | 商标®/™使用规范 | 法律风险 | 未注册标®？ |
| 5 | 字体版权 | 侵权风险 | 商用字体是否有授权？ |

### Layer 1 输出格式

```
📍 Phase 5 · Layer 1 · 快速合规扫描
━━━━━━━━━━━━━━━━━━━
风险分级：
🔴 红旗（必须修改）：{N}项
🟡 黄旗（建议修改）：{N}项
🟢 通过：{N}项

🔴 红旗详情：
• {物料名}：{问题} → 建议：{修正方案}

请确认：
1. 红旗问题是否立即修正？（是 / 我可以接受风险）
2. 是否查看完整合规审查（全部 20 项）？（是 / 否）
```

### P5-G1 · 方向确认门

**红旗必须清零后才能进入 Layer 2。**
用户选择「我可以接受风险」→ 记录风险确认，继续。
用户选择「查看完整审查」→ 进入 Layer 2。

## Layer 2 · 细节层（完整合规审查 + 审美自检）

> **目标：对全部物料执行完整的合规审查和审美质量自检。**
> **仅在用户通过 P5-G1 后解锁。**

### 2.1 完整合规审查

读取 `references/design-compliance.md`，按物料类型执行相关审查项：

| 物料类型 | 必查项 |
|---------|--------|
| 招牌/户外广告 | 中文字号≥英文、不得单独外文、招牌高度、商场规范 |
| 包装 | 刀图（R10）、食品标签区域、GB 7718+GB 28050、声称合规 |
| 海报/宣传册 | 绝对化用语、虚假宣传、字体版权、WCAG对比度 |
| Logo | 商标法§10、®/™规范、最小尺寸、安全空间 |
| 数字物料 | WCAG无障碍、sRGB色彩、Retina适配 |

### 2.2 审美自检（7大维度）

读取 `references/design-aesthetics.md`，对每个产出执行：

| 锚点 | 自检问题 |
|------|---------|
| Logo质量 | 缩小1cm/黑白/反色/3米远/10秒速写 — 是否通过？ |
| 配色质量 | 主色≤3种？去色后层级清晰？对比度≥4.5:1？ |
| 排版质量 | 字体≤3种？行距1.3-1.6倍？所有元素对齐？ |
| 整体质量 | 有明确视觉焦点？5秒传达核心信息？无"AI味"？ |

### 2.3 场景评审侧重（融入huashu-design）

不同物料类型的评审重点不同，不要「一刀切」：

| 物料类型 | 最重要维度 | 次重要 | 可放宽 |
|----------|-----------|--------|--------|
| 海报/户外广告 | 创新性、视觉层级 | 哲学一致性 | 功能性（单图不涉及交互） |
| 信息图/数据海报 | 功能性、视觉层级 | 细节执行 | 创新性（准确优先） |
| PPT/演示文稿 | 视觉层级、功能性 | 细节执行 | 创新性（清晰优先） |
| 画册/白皮书 | 细节执行、功能性 | 视觉层级 | 创新性（专业优先） |
| 包装 | 功能性、细节执行 | 原创识别 | 哲学一致性（可用性优先） |
| 名片 | 细节执行、原创识别 | 视觉层级 | 功能性（信息少） |
| 社交媒体图 | 创新性、视觉层级 | 哲学一致性 | 细节执行（氛围优先） |
| Logo | 原创识别、哲学对齐 | 细节执行 | 功能性（纯标识） |

### 2.4 常见设计问题 Top 10（融入huashu-design）

对每类物料，优先检查这些问题：

| # | 问题 | 判断方法 | 修复（含数值） |
|---|------|---------|--------------|
| 1 | **字号层级不足** | 标题和正文差距 <2.5 倍 | 标题至少为正文的 3 倍（如正文 16px → 标题 48-64px） |
| 2 | **颜色过多** | 使用 5 种以上颜色，没有主次 | 限制为 1 主色 + 1 辅色 + 1 强调色 + 灰阶 |
| 3 | **间距不统一** | 元素间距随意，没有系统 | 建立 8pt 网格系统（间距只用 8/16/24/32/48/64px 或其倍数） |
| 4 | **留白不足** | 所有空间都被内容填满 | 留白至少占总面积 40%（极简风格 60%+） |
| 5 | **字体过多** | 使用 3 种以上字体家族 | 最多 2 种（1 标题 + 1 正文），用字重和大小创造变化 |
| 6 | **对齐不一致** | 有的左对齐，有的居中，有的右对齐 | 选定一种对齐方式全局统一（推荐左对齐） |
| 7 | **装饰大于内容** | 背景图案/渐变/阴影抢了主要内容 | 「如果删掉这个装饰，设计会变差吗？」不会就删 |
| 8 | **AI 科技 cliché** | 渐变圆球、数字雨、蓝色电路板 | 用抽象隐喻替代直白符号（见 `references/anti-ai-slop.md`） |
| 9 | **信息密度与载体不匹配** | 海报塞了 10+ 元素 / 名片放满联系方式 | 海报 1 焦点 + ≤3 辅助信息；名片 ≤4 项信息 |
| 10 | **过度对称** | AI 默认绝对对称，显得呆板 | 用不对称平衡：主体偏左/右，留白在另一侧平衡 |

## Layer 3 · 高级层（深度风险评估 + 整改方案 + 长期合规策略）

> **目标：为高风险项目或法律敏感行业提供深度分析。**
> **仅在用户通过 P5-G2 且主动要求/场景需要时解锁。**

### 深度风险评估

- 每项违规的法律后果等级（提示/罚款/下架/诉讼）
- 整改优先级排序（P0紧急 / P1重要 / P2建议）
- 替代方案建议（如果完全合规会牺牲设计效果）

### 长期合规策略

- 字体授权清单管理建议
- 商标注册时间线建议
- 多地区合规差异（如出海品牌）

### P5-G3 · Phase 验收门

```
📍 Phase 5 完成 · 最终确认
━━━━━━━━━━━━━━━━━━━
审查状态：全部通过 ✅

请确认：
1. 审查报告是否满意？（满意 / 需补充）
2. 是否进入 Phase 6（落地指导+交付）？（是 / 否）
```

---

## 关键提示

- **三重保障**：每个物料生成时预扫描（R9前置）+ 全部完成后总审查（Phase 5）+ 审美质量自检
- 合规审查不是「找茬」，是「保护设计顺利落地」
- 发现违规时，提供具体修正方案，不只说「不合规」
- 用户说「我可以接受风险」→ 记录确认，但明确告知潜在后果

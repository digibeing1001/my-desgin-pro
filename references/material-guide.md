## 二、包装材质与工艺（按品类）

### 2.1 食品包装

| 层级 | 推荐材质 | 推荐工艺 | 参考价格（仅供参考，以实际供应商报价为准） |
|------|---------|---------|----------------------------------|
| 经济型 | 白卡纸 250–300g + PE 覆膜 | 四色胶印 + 哑膜 | ¥0.3–0.8 / 个（中小盒，起订量 1000+） |
| 标准型 | 白卡纸 300–350g + 覆膜 | 四色胶印 + 局部 UV + 哑膜 | ¥0.8–2.0 / 个 |
| 高端型 | 特种纸（触感纸/珠光纸）+ 食品级内衬 | 烫金/烫银 + 凹凸 + 覆膜 | ¥2.0–8.0 / 个 |

**为什么用 UV 局部光油？**
- **优点**：Logo/图案局部高光，触感对比强烈，提升档次；防水防油，适合食品包装
- **局限**：成本比覆膜高约 15–30%
- **参考价**：局部 UV 约 ¥0.1–0.3 / 个（视面积）

**合规注意**：食品包装必须有食品接触材料合规标识、生产日期/保质期区域、营养成分表区域（R9）。

- **参考价**：烫金版费约 ¥200–500（一次性），单价约 ¥0.2–0.5 / 个

## 五、工艺速查表

| 工艺 | 效果 | 适用场景 | 参考加价 |
|------|------|---------|---------|
| **覆亮膜** | 表面光亮、防水 | 海报、包装盒、画册封面 | +10–20% |
| **覆哑膜** | 表面哑光、高级感 | 名片、化妆品盒、高端画册 | +15–25% |
| **局部 UV** | 局部高光凸起、触感对比 | Logo、图案强调、食品/化妆品 | +15–30% |
| **烫金** | 金属光泽、豪华感 | 礼品盒、化妆品、高端名片 | +20–40%（含版费） |
| **烫银** | 冷金属光泽、科技感 | 数码3C、科技品牌 | +20–40% |
| **凹凸** | 立体触感、无油墨 | 名片、礼盒、高端包装 | +10–20% |
| **压纹** | 纹理质感、防滑 | 特种纸包装、封面 | +10–15% |
| **模切** | 异形裁切 | 标签、包装盒、创意物料 | 刀版费 ¥100–500 |

## 六、使用规则

1. **按品类匹配**：先确定用户所属品类，再在该品类表格中选择层级
2. **预算导向**：询问用户预算范围后，在对应层级内推荐最优组合
3. **环保趋势**：2025–2026 年市场趋势偏向可再生/可降解材质（牛皮纸、再生纸、PLA），如用户无特殊要求，可主动推荐环保选项并说明溢价约 10–20%
4. **价格声明**：本指南所有价格为**市场参考价**，实际价格因地区、供应商、起订量、季节波动，**必须以实际供应商报价为准**

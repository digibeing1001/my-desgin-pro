## 一、标志规范

| 规范项 | 钉钉参数 |
|--------|---------|
| 印刷最小尺寸 | 8mm(标准) / 25mm(竖版) |
| 网络最小尺寸 | 40px / 100px |
| 安全空间单位 | X=logo宽度 |
| 标志变体 | 中文+英文+Slogan组合 |
| 颜色优先级 | 蓝>白>黑 |

### 标志禁令（与其他5品牌共同验证的10项中）
- 禁止改变标志比例/拉伸变形
- 禁止改变标志标准色
- 禁止倾斜/旋转标志
- 禁止添加投影/立体化效果
- 禁止改变标志字体
- 禁止随意组合/排列标志元素

## 四、4大设计系统结构

```
A 基础设计系统
├── 标志规范(中文+英文+Slogan)
├── 集团背书标志
├── 字体规范
└── 色彩规范

B 网络基础设计系统
└── 品牌标志规范

C VI应用设计系统
├── 办公事物用品规范
├── 公共关系系统规范
└── R角规范（钉钉特色）

D 广告应用设计系统
├── 传播应用规范
└── 视频应用规范
```

### 钉钉R角规范（特色）
- 钉钉独有的R角(圆角)设计规范，是品牌视觉识别的重要组成部分

# 设计哲学风格库

> 用于平面设计（海报/包装/画册/名片/品牌VI等）的设计风格方向库。
> 每种风格提供：哲学内核 + 核心视觉特征 + 适用场景 + 生图提示词 DNA。
>
> **使用方式**：Phase 2 设计哲学创建时，从本库选取 1-2 个方向作为起点，结合 brand-profile.md 的品牌定位进行适配。不要照搬，要翻译为品牌语言。

---

## 流派速查

| 流派 | 风格 | 核心气质 | 最佳物料 |
|------|------|---------|---------|
| 信息建筑派 | Pentagram / Müller-Brockmann | 理性、克制、数据驱动 | 画册、报告、数据海报、VI手册 |
| 运动诗学派 | Field.io / Locomotive | 动感、沉浸、技术美学 | 科技发布会海报、动态Logo、展览设计 |
| 极简主义派 | Kenya Hara / Build | 留白、秩序、精致 | 奢侈品包装、高端名片、品牌手册 |
| 实验先锋派 | Sagmeister & Walsh | 意外、情感、视觉冲击 | 活动海报、艺术展览、概念包装 |
| 东方哲学派 | Takram / 原研哉 | 温润、诗意、思辨 | 文创品牌、茶/酒包装、东方美学海报 |
| 编辑排版派 | Bloomberg Businessweek / 卫报 | 大胆排版、信息密度、新闻感 | 信息图、杂志风海报、年报 |

## 一、信息建筑派

> 哲学：「信息不是装饰，是建筑材料」

### 1. Pentagram / Michael Bierut 风格

**哲学**：字体即语言，网格即思想

**核心视觉特征**：
- 极度克制的颜色（黑白 + 1 个品牌色）
- 瑞士网格系统的现代演绎
- 字体排印作为主要视觉语言
- 负空间的战略性使用（60%+ 留白）
- 信息层级即视觉结构

**适用场景**：企业画册、年度报告、品牌VI手册、数据海报、专业服务机构品牌

**生图提示词 DNA**：
```
Pentagram / Michael Bierut style graphic design:
- Extreme typographic hierarchy, Helvetica/Univers family
- Swiss grid with precise mathematical spacing
- Black/white + one accent color
- Information architecture as visual structure
- 60%+ whitespace ratio
- Data visualization as primary decoration
- Professional, authoritative, editorial feel
```

---

### 2. Müller-Brockmann 传承 — 瑞士网格纯粹主义

**哲学**：客观性即美

**核心视觉特征**：
- 数学精确的网格系统（8pt 基线）
- 绝对的左对齐或居中
- 单色或双色方案（黑 + 一个 accent）
- 功能主义至上
- Akzidenz-Grotesk 或类似理性字体

**适用场景**：博物馆海报、音乐会海报、学术出版物、极简品牌系统

**生图提示词 DNA**：
```
Josef Müller-Brockmann Swiss modernism:
- Mathematical grid system
- Strict alignment (flush left or centered)
- Two-color maximum (black + one accent)
- Rationalist typeface
- No decorative elements
- Timeless, objective aesthetic
- Concert poster / museum poster feel
```

## 二、运动诗学派

> 哲学：「技术本身就是一种流动的诗」

### 3. Field.io — 算法美学

**哲学**：代码即设计师

**核心视觉特征**：
- 生成艺术系统（抽象几何）
- 抽象几何的智能编排
- 技术感与艺术性的平衡
- 单色基底 +  vibrant accent
- Voronoi 图或 Delaunay 三角化纹理

**适用场景**：科技品牌发布会海报、音乐节海报、数字艺术展览、概念包装设计

**生图提示词 DNA**：
```
Field.io generative design style:
- Abstract geometric patterns, algorithmically generated
- Dynamic composition that feels computational
- Monochromatic base with vibrant accent
- Mathematical precision in spacing
- Voronoi diagrams or Delaunay triangulation
- Clean, code-inspired aesthetic
- Digital art exhibition poster feel
```

---

### 4. Locomotive — 滚动叙事

**哲学**：观看不是浏览，是旅程

**核心视觉特征**：
- 电影化的分镜叙事
- 大胆的空间留白
- 动态元素的精确编排
- 深色模式（近黑背景）
- 战略性发光 accent

**适用场景**：电影海报、大型活动主视觉、展览空间设计、沉浸式品牌体验

**生图提示词 DNA**：
```
Locomotive scroll narrative style:
- Film-like scene composition with depth
- Generous spacing between elements
- Bold typography emerging from darkness
- Dark mode (near-black backgrounds)
- Strategic glowing accents
- Hero sections with dramatic scale
- Cinematic poster feel
```

## 三、极简主义派

> 哲学：「删减到无法再删」

### 5. Kenya Hara（原研哉）— 空的设计

**哲学**：设计不是填充，是清空

**核心视觉特征**：
- 极致的留白（80%+）
- 纸张质感的数字化表达
- 白色的层次（暖白、冷白、米白）
- 触觉的视觉化
- 极少色彩（若有，极淡）

**适用场景**：日本/东方美学品牌、高端生活方式包装、艺术画册、品牌手册、无印良品式品牌

**生图提示词 DNA**：
```
Kenya Hara "emptiness" design:
- Extreme whitespace (80%+)
- Paper texture and tactility
- Layers of white (warm white, cool white, off-white)
- Minimal color (if any, very desaturated)
- Design by subtraction not addition
- Zen simplicity
- Japanese aesthetic, Muji-inspired
```

---

### 6. Build — 当代极简品牌

**哲学**：精致的简单比复杂更难

**核心视觉特征**：
- 奢侈品级的留白（70%+）
- 微妙的字重对比（200-600）
- 单一强调色的战略使用
- 呼吸感的节奏
- 高端产品摄影美学
- 黄金比例构图

**适用场景**：奢侈品包装、高端品牌名片、时尚画册、精品酒店品牌

**生图提示词 DNA**：
```
Build studio luxury minimalism:
- Generous whitespace (70%+ of area)
- Subtle typography weight shifts
- Single accent color used sparingly
- High-end product photography aesthetic
- Soft shadows and subtle gradients
- Golden ratio proportions
- Luxury branding, refined elegance
```

## 四、实验先锋派

> 哲学：「打破规则即创造规则」

### 7. Sagmeister & Walsh — 快乐极简

**哲学**：美即功能的情感维度

**核心视觉特征**：
- 意外的色彩爆发（在极简基底上）
- 手工感与数字的融合
- 正能量的视觉语言
- 实验性但可读
- 通过不完美传递人性温暖

**适用场景**：创意机构品牌、文化活动海报、概念包装、艺术展览、年轻品牌

**生图提示词 DNA**：
```
Sagmeister & Walsh joyful philosophy:
- Unexpected color bursts on minimal base
- Handmade elements mixed with digital precision
- Optimistic visual language
- Experimental typography that remains legible
- Human warmth through imperfection
- Mix of analog and digital aesthetics
- Creative, bold, unexpected
```

---

### 8. Ash Thorp — 赛博诗意

**哲学**：未来不是冰冷的，是孤独的诗

**核心视觉特征**：
- 电影级的光影
- 赛博朋克的温暖版本（橙/青，非冷蓝）
- 故事性的概念设计
- 工业美学的精致化
- 体积光和神光效果

**适用场景**：科幻电影海报、游戏品牌、科技发布会、概念艺术、潮牌设计

**生图提示词 DNA**：
```
Ash Thorp cinematic concept art:
- Film-grade lighting and atmospheric effects
- Warm cyberpunk (orange/teal, NOT cold blue)
- Industrial design meets luxury
- Narrative concept art feel
- Volumetric lighting and god rays
- Blade Runner warmth over Tron coldness
- Futuristic yet poetic
```

## 五、东方哲学派

> 哲学：「留白即内容」

### 9. Takram — 日式思辨设计

**哲学**：技术是思考的媒介

**核心视觉特征**：
- 概念原型的优雅
- 柔和的科技感（圆角、柔和阴影）
- 图表即艺术
- 谦逊的精致
- 中性自然色（米色、柔灰、淡绿）

**适用场景**：日本品牌、科技+人文交叉领域、文创产品、概念设计、思辨性品牌

**生图提示词 DNA**：
```
Takram Japanese speculative design:
- Elegant concept prototypes and diagrams
- Soft tech aesthetic (rounded corners, gentle shadows)
- Charts and diagrams as art pieces
- Modest sophistication
- Neutral natural colors (beige, soft gray, muted green)
- Design as philosophical inquiry
- Japanese design sensibility
```

---

### 10. Neo Shen — 东方光影诗

**哲学**：技术需要人的温度

**核心视觉特征**：
- 水墨晕染的数字化
- 柔和的光晕效果
- 诗意的留白
- 情感化的色彩（深蓝、暖灰、柔金）
- 书法影响的字体排印
- 大气深度

**适用场景**：中国品牌、茶/酒/文化产品包装、东方美学海报、博物馆文创、传统品牌焕新

**生图提示词 DNA**：
```
Neo Shen poetic Chinese aesthetic:
- Digital interpretation of ink wash painting
- Soft glow and light diffusion effects
- Poetic negative space
- Emotional palette (deep blues, warm grays, soft gold)
- Calligraphic influences in typography
- Atmospheric depth
- Chinese ink painting meets modern design
```

## 六、编辑排版派

> 哲学：「排版即内容」

### 11. Bloomberg Businessweek / 杂志编辑风

**哲学**：信息密度本身可以很美

**核心视觉特征**：
- 大胆的字体排版（超大标题）
- 高信息密度但不混乱
- 数据与视觉的融合
- 新闻纸/印刷质感
- 强烈的视觉节奏（大图 + 密集文字交替）

**适用场景**：信息图、年度报告、杂志风海报、数据可视化、新闻品牌

**生图提示词 DNA**：
```
Bloomberg Businessweek editorial style:
- Bold typographic headlines dominating composition
- High information density without clutter
- Data integrated with visual design
- Newsprint/print texture feel
- Strong visual rhythm (large image + dense text)
- Magazine editorial spread aesthetic
- Data tells a story
```

## 七、粗野主义派

> 哲学：「诚实比精致更重要」

### 12. Experimental Jetset — 概念极简

**哲学**：一个想法 = 一个形式

**核心视觉特征**：
- 单一视觉隐喻贯穿整个设计
- 蓝/红/黄 + 黑白的蒙德里安色系
- 字体即图形
- 反商业的诚实设计
- 网格基础上故意打破规则

**适用场景**：独立品牌、音乐/艺术海报、概念设计、反叛品牌、文化机构

**生图提示词 DNA**：
```
Experimental Jetset conceptual minimalism:
- Single visual metaphor for entire design
- Primary colors (red/blue/yellow) + black/white
- Typography as main graphic element
- Grid-based with deliberate rule-breaking
- No photography, only type and geometry
- Anti-commercial, honest aesthetic
- Bold, conceptual, raw
```

## 八、自然有机派

> 哲学：「真实比完美更有说服力」

### 13. Aesop / 无印良品 — 有机真实

**哲学**：材料即信息

**核心视觉特征**：
- 大地色系（赭石、鼠尾草、亚麻）
- 自然纹理（纸张、木材、石材）
- 手写/手绘元素
- 不规则的有机形状
- 可持续/环保的视觉暗示

**适用场景**：有机产品包装、生活方式品牌、环保品牌、天然护肤品、手作品牌

**生图提示词 DNA**：
```
Aesop / Muji organic authenticity:
- Earth tone palette (terracotta, sage, linen)
- Natural textures (paper, wood, stone)
- Handwritten/hand-drawn elements
- Irregular organic shapes
- Sustainable/eco-friendly visual cues
- Real materials, not glossy perfection
- Warm, honest, tactile
```

## 使用指南

### 如何选择风格

1. **看品牌定位**：奢侈品 → 极简主义派；科技 → 信息建筑派/运动诗学派；文创 → 东方哲学派/自然有机派
2. **看受众**：B2B 专业客户 → 信息建筑派；Z 世代消费者 → 实验先锋派；高端消费者 → 极简主义派
3. **看物料类型**：画册/报告 → 信息建筑派；海报/活动 → 实验先锋派/编辑排版派；包装 → 极简/自然有机/东方

### 生图提示词组合公式

```
[风格提示词DNA] + [物料类型描述] + [品牌内容描述] + [技术参数]
```

**示例**：
```
Kenya Hara "emptiness" design:
- Extreme whitespace, layers of white
- Minimal color, design by subtraction

Tea brand packaging design:
- Premium loose-leaf tea box
- Japanese/Chinese fusion aesthetic
- Single product focus on front panel

High resolution, print-ready, clean layout,
professional graphic design, meticulously crafted
```

### Mood, Not Layout 原则

> **描述情绪而非布局**。

| 杀死多样性的写法 | 激发创造力的写法 |
|----------------|----------------|
| "标题居中，图片右侧，下方三列" | "warm like Sunday morning, with the calm confidence of a master tea maker" |
| "颜色比例 60% 蓝 / 30% 白 / 10% 金" | "deep ocean blue with occasional gold leaf accents, like traditional Chinese lacquerware" |
| "使用 Helvetica，16pt 正文" | "Swiss precision meets Eastern meditation" |

**短提示词 > 长提示词**。描述 3 句情绪和内容，比 30 行布局细节效果更好。

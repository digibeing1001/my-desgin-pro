## 1. Logo设计5大原则

> 来源：Paul Rand（IBM/UPS/ABCLogo设计者）+ Sagi Haviv（Chase/MobilLogo设计者）

### 1.1 五原则速查

| 原则 | 核心要求 | 自检问题 | 典型反面案例 |
|------|---------|---------|-------------|
| **简洁性 (Simplicity)** | 元素越少越好，一个核心概念 | 去掉任何一个元素还成立吗？ | 2012伦敦奥运会Logo — 过于复杂 |
| **易记性 (Memorability)** | 第一眼就能记住大致形状 | 遮住一半还能认出来吗？ | 各种"字母+地球+翅膀"组合Logo |
| **永恒性 (Timelessness)** | 不追逐潮流，5-10年后仍不过时 | 去掉当前流行元素还成立吗？ | 2000年代所有带"光泽/3D"的Logo |
| **通用性 (Versatility)** | 任何尺寸/背景/媒介都能用 | 缩到16px图标还清晰吗？黑底白底都好看？ | 含复杂渐变的Logo — 缩小模糊 |
| **适配性 (Appropriateness)** | 视觉风格与品牌定位一致 | 看到Logo能猜到什么行业吗？ | 律师所用卡通Logo |

### 1.2 好Logo的黄金测试

```
测试1：缩小测试 — 缩小到1cm宽，是否仍可识别？
测试2：传真测试 — 用黑白传真发送，是否仍可识别？
测试3：反色测试 — 白底黑字↔黑底白字，是否都好看？
测试4：远看测试 — 3米外看，是否仍能认出？
测试5：速写测试 — 能否在10秒内凭记忆画出大致形状？
```

### 1.3 Logo设计流程参考

1. **调研**：品牌定位、行业视觉分析、竞品Logo拆解
2. **概念**：1个核心概念（不是5个概念的拼凑）
3. **草图**：20-30个手绘草图，快速筛选到3个
4. **精修**：矢量化、色彩、变体设计
5. **测试**：执行黄金测试5项
6. **规范**：最小尺寸、安全空间、禁用示例

## 2. 格式塔7大视觉原则

> 来源：Max Wertheimer/Kurt Koffka/Wolfgang Kohler，格式塔心理学派

这些原则解释了人眼如何自动组织视觉元素。违反这些原则 = 视觉混乱。

| 原则 | 含义 | 设计应用 | 违反后果 |
|------|------|---------|---------|
| **接近性 (Proximity)** | 距离近的元素被视为一组 | 相关信息靠近放置，不相关的拉开距离 | 信息关系混乱，用户无法快速理解分组 |
| **相似性 (Similarity)** | 相似的元素被视为一组 | 同级信息用相同颜色/大小/形状 | 层级模糊，用户无法区分主次 |
| **连续性 (Continuity)** | 眼睛倾向于沿路径移动 | 利用线条/对齐创造视觉动线 | 视觉跳跃，阅读不流畅 |
| **闭合性 (Closure)** | 大脑自动补全不完整的形状 | Logo可适当省略细节，让观者参与 | 过于破碎的图形无法被识别 |
| **图形-背景 (Figure-Ground)** | 人眼自动区分前景和背景 | 确保主体突出，背景退后 | 主次不分，视觉疲劳 |
| **共同命运 (Common Fate)** | 同步运动的元素被视为一组 | 动态设计中同向运动的元素被视为关联 | 动画缺乏组织感 |
| **对称性 (Symmetry)** | 对称的元素被视为整体 | 对称构图带来稳定/正式感 | 不对称需刻意平衡，否则显得随意 |

### 实操指南

- **名片**：接近性 → 联系方式归组，姓名单独突出
- **海报**：连续性 → 用对齐创造Z型/F型阅读路径
- **Logo**：闭合性 → 不要填满所有细节，留想象空间
- **画册**：相似性 → 同级标题用统一风格，不同级别用不同风格

## 3. Itten色彩理论

> 来源：Johannes Itten，《色彩艺术》(1961)，包豪斯学院

### 3.1 色彩对比7类型

| 对比类型 | 说明 | 设计应用 |
|----------|------|---------|
| **色相对比** | 互补色并置产生强烈对比 | CTA按钮、强调信息 |
| **明暗对比** | 明度差异产生层次 | 前景/背景区分、视觉层级 |
| **冷暖对比** | 暖色前进、冷色后退 | 创造空间感、引导视线 |
| **补色对比** | 色轮对面的颜色 | 最高冲击力、标志色组合 |
| **同时对比** | 同一灰色在不同背景下色感不同 | 慎用灰色，注意背景影响 |
| **面积对比** | 大面积色vs小面积色 | 60-30-10法则（主色60%/辅色30%/强调色10%） |
| **纯度对比** | 高饱和vs低饱和 | 高饱和=焦点，低饱和=背景 |

### 3.2 色彩心理学（品牌设计核心参照）

| 颜色 | 正面联想 | 负面联想 | 适用行业 | 典型品牌 |
|------|---------|---------|---------|---------|
| **红色** | 激情/活力/紧急/食欲 | 危险/攻击/冲动 | 餐饮/促销/体育 | 可口可乐/肯德基/Netflix |
| **橙色** | 创意/热情/友好/年轻 | 廉价/轻浮 | 创意/电商/儿童 | Fanta/亚马逊/淘宝 |
| **黄色** | 乐观/温暖/注意力/智慧 | 警告/焦虑/怯懦 | 儿童/外卖/快消 | 麦当劳/美团/宜家 |
| **绿色** | 自然/健康/成长/财富 | 嫉妒/不成熟 | 有机/医疗/金融 | 星巴克/支付宝/微信 |
| **蓝色** | 信任/专业/科技/冷静 | 冷漠/疏远/忧郁 | 科技/金融/医疗 | IBM/支付宝/中国移动 |
| **紫色** | 奢华/创意/神秘/灵性 | 做作/不务实 | 美妆/奢侈品/教育 | 京东/雅诗兰黛 |
| **粉色** | 温柔/浪漫/女性/甜蜜 | 幼稚/脆弱 | 美妆/母婴/甜品 | 芭比/Victoria's Secret |
| **黑色** | 高端/权威/优雅/力量 | 死亡/压抑/疏远 | 奢侈品/科技/时尚 | Chanel/Apple/耐克 |
| **白色** | 纯净/简洁/现代/空间 | 空虚/冰冷/无聊 | 科技/医疗/极简品牌 | Apple/无印良品 |
| **金色** | 奢华/尊贵/成就/传统 | 俗气/炫耀 | 金融/奢侈品/传统 | LV/中国银行 |

### 3.3 配色实用法则

| 法则 | 内容 | 示例 |
|------|------|------|
| **60-30-10法则** | 主色60% + 辅色30% + 强调色10% | 蓝色底+白色字+红色CTA |
| **3色原则** | 一个画面不超过3种主色 | 超过3种 = 混乱 |
| **黑白灰打底** | 先用黑白灰确定层级，再加色彩 | 确保去掉色彩后层级依然清晰 |
| **色彩提取法** | 从品牌参考图中提取3-5种色 | 用color-palette技能自动提取 |

## 4. 排版网格系统

> 来源：Josef Muller-Brockmann，《平面设计中的网格系统》(1981)

### 4.1 网格类型

| 网格 | 结构 | 适用场景 |
|------|------|---------|
| **单栏网格** | 1列居中 | 信纸/证书/邀请函 |
| **双栏网格** | 2列等宽或左宽右窄 | 画册/折页/宣传单 |
| **三栏网格** | 3列等宽 | 报纸/杂志/网页 |
| **模块网格** | 行列交叉的格子 | 画册内页/产品目录 |
| **基线网格** | 水平线对齐文字基线 | 正文密集的出版物 |
| **自由网格** | 打破网格的刻意留白 | 创意海报/艺术画册 |

### 4.2 排版参数速查

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| **行距** | 字号的1.3-1.6倍 | 中文1.4倍，英文1.5倍 |
| **段间距** | 字号的1.5-2倍 | 明显大于行距，区分段落 |
| **字间距** | 默认或微紧(-0.02em) | 中文不需调整，英文标题可微紧 |
| **栏间距** | 字号的2-3倍 | 太窄=视觉粘连，太宽=失去关联 |
| **页边距** | 内边距<外边距，上边距<下边距 | 对页排版（书籍） |
| **字号层级** | 标题:正文:注释 = 2:1:0.75 | 如 28pt:14pt:10pt |

### 4.3 对齐方式选择

| 对齐 | 适用 | 不适用 |
|------|------|--------|
| **左对齐** | 正文/列表/大量文字 | 居中排版的海报 |
| **居中对齐** | 标题/邀请函/证书 | 大段正文（边缘参差不齐） |
| **右对齐** | 创意排版/价格表 | 正文阅读 |
| **两端对齐** | 书籍/杂志正文 | 短文本（出现难看的间距） |

## 5. 视觉层级构建

> 来源：Don Norman，《设计心理学》+ Robin Williams，《写给大家看的设计书》

### 5.1 四大基本原则（CRAP）

| 原则 | 英文 | 说明 | 反面 |
|------|------|------|------|
| **对比** | Contrast | 不同层级之间要有明显差异，不大则完全不同 | 所有文字一样大/一样粗 |
| **重复** | Repetition | 重复使用相同的设计元素建立统一感 | 每页用不同字体/颜色 |
| **对齐** | Alignment | 每个元素与页面上至少一个其他元素有视觉关联 | 元素随意摆放 |
| **亲密** | Proximity | 相关项目靠近，不相关项目远离 | 所有信息等距排列 |

### 5.2 视觉动线（阅读路径）

| 布局 | 动线 | 适用场景 |
|------|------|---------|
| **Z型** | 左上→右上→左下→右下 | 网页/横版海报/名片 |
| **F型** | 水平扫描→左侧垂直扫描 | 文字密集页面/新闻列表 |
| **居中型** | 中心焦点→放射外扩 | 海报/封面/社交媒体图 |
| **对角线型** | 左上→右下对角 | 动感设计/运动品牌 |

### 5.3 层级构建方法（按效果从强到弱）

1. **大小**：最大的最先被看到
2. **色彩/对比**：高对比/高饱和最先被看到
3. **位置**：左上角和中心最先被看到
4. **形状**：不规则形状最先被看到
5. **留白**：周围留白最多的最先被看到

## 7. 审美质量自检清单

> 每次设计完成后，对照此清单逐项检查。如果你是设计小白，这份清单就是你的"审美防线"。

### A. Logo自检

- [ ] 缩小到1cm宽仍可识别
- [ ] 黑白版本清晰
- [ ] 反色（白底↔黑底）都好看
- [ ] 3米远看仍可辨认
- [ ] 10秒内能凭记忆画出大致形状
- [ ] 没有使用超过2种主色
- [ ] 没有渐变/投影/3D效果

### B. 配色自检

- [ ] 主色不超过3种（60-30-10法则）
- [ ] 去掉色彩后，层级依然清晰（明暗层级测试）
- [ ] 文字与背景对比度足够（WCAG ≥ 4.5:1）
- [ ] 没有红绿配色作为唯一区分手段
- [ ] 色彩与品牌定位一致（参考色彩心理学表）

### C. 排版自检

- [ ] 字体不超过3种
- [ ] 行距是字号的1.3-1.6倍
- [ ] 所有元素都对齐到某条线
- [ ] 相关信息靠近（亲密性原则）
- [ ] 有清晰的视觉动线（Z型/F型/居中型）
- [ ] 最重要的信息最大/最醒目

### D. 整体自检

- [ ] 有明确的视觉焦点（第一眼看到什么？）
- [ ] 5秒内能传达核心信息
- [ ] 与品牌定位/行业特征一致
- [ ] 不同物料之间风格统一
- [ ] 没有"什么都有但什么都不突出"的感觉
- [ ] 看起来不像是AI生成的（避免"AI味"：过度对称、无意义装饰、文字乱码）

## 8. 设计评审评分标准（0-10分制）

> 融入 huashu-design 五维评审评分细则，为 `phase-5.md` 五维专家审查提供量化参照。

### 8.1 哲学一致性（Philosophy Alignment）

| 分数 | 标准 |
|------|------|
| 9-10 | 设计完美体现品牌哲学的核心精神，每个元素都有哲学依据 |
| 7-8 | 整体方向正确，核心特征到位，个别细节偏离 |
| 5-6 | 能看出意图，但执行时混入了其他风格元素，不够纯粹 |
| 3-4 | 仅在表面模仿，未理解哲学内核 |
| 1-2 | 与品牌哲学基本无关 |

**评审要点**：
- 是否使用了品牌哲学中定义的标志性手法？
- 色彩、字体、布局是否符合该哲学体系？
- 有没有「自相矛盾」的元素？（如选了 Kenya Hara 的空却塞满内容）

### 8.2 视觉层级（Visual Hierarchy）

| 分数 | 标准 |
|------|------|
| 9-10 | 视线自然沿设计者意图流动，信息获取零摩擦 |
| 7-8 | 主次关系清晰，偶有 1-2 处层级模糊 |
| 5-6 | 能分出标题和正文，但中间层级混乱 |
| 3-4 | 信息平铺，没有明确的视觉入口 |
| 1-2 | 混乱，不知道先看哪里 |

**评审要点**：
- 标题与正文的字号对比是否足够？（至少 2.5 倍）
- 颜色/粗细/大小是否建立了 3-4 个清晰层级？
- 留白是否在引导视线？
- **眯眼测试**：眯起眼看，层级是否仍然清晰？

### 8.3 工艺质量（Craft Quality）

| 分数 | 标准 |
|------|------|
| 9-10 | 像素级精确，对齐、间距、颜色无任何瑕疵 |
| 7-8 | 整体精致，有 1-2 处微小对齐/间距问题 |
| 5-6 | 基本对齐，但间距不统一，颜色使用不够系统 |
| 3-4 | 明显的对齐错误、间距混乱、颜色过多 |
| 1-2 | 粗糙，看起来像草稿 |

**评审要点**：
- 是否使用了统一的间距系统？
- 同类元素的间距是否一致？
- 颜色数量是否受控？（通常不超过 3-4 种）
- 字体家族是否统一？（通常不超过 2 种）
- 边缘对齐是否精确？

### 8.4 功能完整（Functionality）

| 分数 | 标准 |
|------|------|
| 9-10 | 每个元素都服务于目标，零冗余 |
| 7-8 | 功能导向明确，有少量可删减的装饰 |
| 5-6 | 基本可用，但有明显的装饰性元素分散注意力 |
| 3-4 | 形式大于功能，需要努力寻找信息 |
| 1-2 | 完全被装饰淹没，失去了传达信息的能力 |

**评审要点**：
- 删掉任何一个元素，设计会变差吗？（如果不会，就应该删）
- 关键信息是否在最显眼的位置？
- 是否有「因为好看所以加上去」的元素？
- 信息密度与载体是否匹配？

### 8.5 原创识别（Originality）

| 分数 | 标准 |
|------|------|
| 9-10 | 令人耳目一新，在品牌框架内找到了独特表达 |
| 7-8 | 有自己的想法，不是简单的模板套用 |
| 5-6 | 中规中矩，看起来像模板 |
| 3-4 | 大量使用了 cliché（如渐变圆球代表 AI） |
| 1-2 | 完全是模板或素材拼凑 |

**评审要点**：
- 是否避免了常见 cliché？（见 `references/anti-ai-slop.md`）
- 在遵循设计哲学的同时是否有品牌独特表达？
- 是否有「意想不到但很合理」的设计决策？

## 9. 快速测试工具

### 3秒测试
> 把设计拿给不熟悉项目的人看 3 秒，然后移开。问：
> 1. 你第一眼看到什么？
> 2. 这是什么品牌/产品？
> 3. 它想传达什么情绪？

如果三个问题有任何一个答不上来 → 视觉层级或品牌识别有问题。

### 眯眼测试
> 眯起眼睛看设计（或把图片缩小到 100px 宽）。

- 层级是否仍然清晰？
- 视觉焦点是否仍在原位置？
- 是否有「跳出来」的突兀元素？

### 去色测试
> 把设计转成黑白/灰度。

- 信息层级是否依然可读？
- 是否有仅靠颜色区分的重要信息？（色盲用户会丢失）
- 明暗对比是否足够？

### 反色测试
> 把设计反色（黑底→白底，白底→黑底）。

- Logo 是否仍然清晰？
- 关键信息是否仍然可读？
- 是否有仅在特定背景色下才成立的元素？

| 要素 | 说明 |
|------|------|

| **折线（蓝色虚线）** | 折叠位置线 |

| **色值标注** | 关键色块的CMYK值 |
| **图文位置** | 品牌元素在展开图上的精确位置 |

## H. Logo规范刚性要求

> 以下为VI基础系统中Logo的强制规范，任何品牌VI项目都必须产出。

### H1. Logo最小使用尺寸

| 载体 | 最小尺寸 | 说明 |
|------|---------|------|
| 印刷品 | Logo宽度 ≥ 15mm | 低于此尺寸细节丢失 |
| 手机端 | Logo宽度 ≥ 36px | 确保Retina屏清晰 |
| Favicon | 16×16px / 32×32px | 仅使用纯图标版 |
| 社交头像 | 110×110px 显示区域 | 图标版优先 |

### H2. Logo安全空间

| 参数 | 要求 | 说明 |
|------|------|------|
| **四周留白** | 安全空间 = Logo高度的1/2 | Logo与任何其他元素的最小距离 |
| **视觉测量** | 以Logo中最高/最低/最左/最右点为界 | 非文字基线，而是视觉边界 |
| **背景色变化** | 安全空间内不得有任何文字/图形/线条 | 包括背景纹理 |

### H3. Logo禁用示例

> 每套VI必须包含以下禁用示意图，防止错误使用。

| 编号 | 禁用类型 | 说明 |
|------|---------|------|
| X01 | **拉伸变形** | 不得非等比缩放Logo |
| X02 | **旋转倾斜** | 不得旋转或倾斜Logo |
| X03 | **添加投影** | 不得添加投影/发光/3D效果 |
| X04 | **添加描边** | 不得为Logo添加描边/轮廓 |
| X05 | **更换颜色** | 不得使用非标准色 |
| X06 | **置于复杂背景** | 不得将Logo放在干扰识别的背景上 |
| X07 | **元素侵入安全空间** | 其他元素不得进入安全空间 |
| X08 | **小于最小尺寸** | 不得使用低于最小尺寸的Logo |
| X09 | **仅用部分** | 不得裁切Logo只显示部分（图标版除外） |
| X10 | **添加文字** | 不得在Logo上叠加文字 |

### H4. 响应式Logo系统

> 数字时代Logo需在不同设备上自适应，每个VI项目至少提供以下版本：

| 版本 | 尺寸范围 | 构成 | 使用场景 |
|------|---------|------|---------|
| **完整版** | ≥ 200px宽 | 图标 + 品牌名 + tagline | 网站/海报/大尺寸 |
| **标准版** | 60-200px宽 | 图标 + 品牌名 | 名片/信纸/常规场景 |
| **紧凑版** | 36-60px宽 | 图标 + 缩写 | 移动端导航/小空间 |
| **图标版** | ≤ 36px | 仅图标 | Favicon/应用图标/头像 |
| **反白版** | 任意 | 完整版/标准版 + 深色背景 | 暗色模式/深色包装 |
| **单色版** | 任意 | 纯黑或纯白 | 传真/报纸/单色印刷 |

### H5. 色彩系统标注规范

> 品牌色彩必须同时标注以下色值：

| 用途 | 色值类型 | 说明 |
|------|---------|------|
| 屏幕显示 | RGB + HEX | 网页/App/H5 |
| 四色印刷 | CMYK | 画册/名片/传单 |
| 专色印刷 | Pantone | 高端印刷/包装 |
| 渐变/辅助 | HSL | 便于衍生色计算 |

**色彩系统组成（每套VI必须）**：

| 组件 | 数量 | 说明 |
|------|------|------|
| 主色 | 1-2种 | 品牌识别核心色 |
| 辅助色 | 2-3种 | 丰富视觉层次 |
| 中性色 | 3-4种 | 黑/白/灰梯度，用于文字/背景 |
| 功能色 | 3种 | 成功(绿)/警告(橙)/错误(红) |
| 禁用色域 | 明确范围 | 哪些明度/饱和度不可使用 |

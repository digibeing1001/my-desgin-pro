## 使用说明

**AI agent 使用方式**：
- 向用户**推荐**这些资源链接
- 通过 `FetchURL` 访问其中的内容（如设计原则文章）
- 通过 `WebSearch` 在这些平台内搜索特定内容
- **不能直接下载**平台内的全部资源（版权/规模限制）

**用户自助方式**：
- 直接访问链接浏览/下载
- 在平台内搜索特定素材

## 资源使用优先级（AI agent）

```
1. 品牌资产协议（brand-spec.md）→ 真实品牌资产
2. 核心资产协议搜索（官网/press kit/官方社媒）→ 官方素材
3. 本索引中的免费资源（Unsplash/Pexels/Wikimedia）→ 通用素材
4. 付费素材平台（Shutterstock/Adobe Stock）→ 告知用户需购买
5. AI 生成（nano-banana-pro 等）→ 以官方参考为基底
```

> **原则**：优先用真实资产，次选免费资源，最后才是 AI 生成。每次使用外部资源都需确认版权状态。

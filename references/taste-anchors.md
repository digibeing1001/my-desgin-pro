# 品位锚点：Graphic Design Pro 审美判断框架

> 来源：融入 huashu-design「品位锚点」方法论，适配平面设计场景。
> **当 AI 缺乏审美判断时，以本文档作为「品位指南针」。**

---

## 核心原则

> **品位 = 一个细节做到 120%，其它做到 80%**
>
> —— 不是所有地方都精致，而是在**合适的地方**足够精致。

这个原则的量化版：
- 80% = 行业合格线（对齐正确、间距统一、色彩和谐、信息清晰）
- 120% = 品牌记忆点（一个让人「值得截图」的质感细节）

**反例**：海报上每个元素都用渐变+阴影+描边+发光 = 处处 100% = 处处平淡 = 没有记忆点。

## 一、字体配对策略

### 基本原则

不要用系统默认字体作 display（标题/品牌名）。**Display 字体负责「风格签名」，Body 字体负责「阅读舒适」**。

### 西文配对（按风格）

| 风格方向 | Display | Body | 气质 |
|---------|---------|------|------|
| Editorial 编辑感 | Playfair Display, Cormorant Garamond | Lato, Source Sans Pro | 知识分子、杂志感 |
| Minimal 极简 | Helvetica Now, Neue Haas Grotesk | Inter, SF Pro | 冷静、理性、科技 |
| Technical 技术感 | JetBrains Mono, Space Mono | Open Sans, Roboto | 数据、代码、精准 |
| Luxury 奢华 | Bodoni, Didot | Cormorant, Lora | 时尚、珠宝、高端 |
| Playful 活泼 | Syne, Bricolage Grotesque | Nunito, Quicksand | 年轻、创意、友好 |
| Brutalist 粗野 | Arial Black, Franklin Gothic | Courier, IBM Plex Mono | 反叛、原始、力量 |

### 中文配对（按风格）

| 风格方向 | Display | Body | 气质 |
|---------|---------|------|------|
| 国潮/传统 | 站酷高端黑, 方正清刻本悦宋 | 思源黑体, 方正兰亭黑 | 东方、书卷、文化 |
| 现代/科技 | 思源黑体 Heavy, 方正兰亭黑 | 思源黑体 Regular, 苹方 | 干净、专业、克制 |
| 温暖/生活 | 站酷快乐体, 造字工房悦圆 | 思源黑体, 阿里巴巴普惠体 | 亲和、日常、轻松 |
| 奢华/高端 | 方正风雅楷宋, 方正清刻本悦宋 | 思源宋体, 方正书宋 | 典雅、精致、东方 |
| 活泼/年轻 | 站酷庆科黄油体, 优设标题黑 | 思源黑体, OPPO Sans | 活力、创意、跳跃 |

### 配对原则

1. **对比度足够**：衬线 vs 无衬线、粗 vs 细、曲 vs 直
2. **情绪一致**：不要「Display 很活泼 + Body 很严肃」
3. **最多 2 种字体家族**：用字重（Light/Regular/Bold）和大小创造变化，不是用更多字体
4. **中文优先**：如果品牌主要受众是中文用户，先定中文字体，再配西文字体

## 二、色彩策略

### 单 accent 贯穿

**最安全的策略**：一个有温度的底色 + **单个** accent 色贯穿全场。

| 底色温度 | 推荐 accent | 适用行业 |
|---------|------------|---------|
| 暖白/米白 (#F5F0E8) | Rust 橙 (#C27558) | 餐饮/生活方式/文创 |
| 冷白 (#F8F9FA) | 墨绿 (#2D5A4A) | 科技/健康/有机 |
| 暖灰 (#EDE8E0) | 深红 (#8B2635) | 奢侈品/酒类/传统 |
| 冷灰 (#E8EAED) | 钴蓝 (#2B5F8A) | 金融/科技/专业 |
| 黑色 (#1A1A1A) | 金色 (#C9A227) | 奢侈品/高端/时尚 |

### 配色禁区

- ❌ 多色聚类（除非数据真的有 ≥3 个分类维度）
- ❌ 凭空发明颜色（参考「反 AI Slop 清单」）
- ❌ 用渐变替代层级（渐变是装饰，不是信息层级工具）
- ❌ 夜间模式随手加反色（不想做就别做）

### oklch 色彩（数字物料）

如果涉及数字物料（网页配图/社交媒体图），用 oklch() 定义色彩：

```css
:root {
  --primary: oklch(0.65 0.18 25);      /* 温暖的 terracotta */
  --primary-light: oklch(0.85 0.08 25); /* 同色系浅色 */
  --primary-dark: oklch(0.45 0.20 25);  /* 同色系深色 */
}
```

oklch 能保证调整亮度时色相不漂移，比 hsl 好用。

## 三、信息密度分型

不同物料需要不同的信息密度策略。**克制不是普适律**。

### 克制型（默认）

**适用**：品牌海报、奢侈品包装、艺术画册、高端名片、品牌手册封面

**原则**：
- 少一层容器
- 少一个 border
- 少一个**装饰性** icon
- 给内容留气口

**量化参考**：
- 海报：1 个视觉焦点 + ≤3 个辅助信息
- 名片正面：姓名 + 职位 + 品牌名 + 1 种联系方式
- 包装正面：品牌名 + 产品名 + 1 个卖点 + 视觉焦点
- 留白占比：≥40%（极简风格 60%+）

### 高密度型（例外）

**适用**：产品说明书、超市促销海报、展会宣传单页、数据报告封面、电商详情页

**原则**：
- 产品核心卖点需要信息密度支撑时，加法优先于克制
- 加的是**有内容的**密度，不是装饰
- 每个新增元素都必须 carry 信息

**量化参考**：
- 促销海报：主标题 + 副标题 + 促销信息 + 时间/地点 + 行动号召 + 视觉焦点
- 产品说明书：分层展示（标题层→核心参数层→详细说明层→联系方式层）
- 信息图：标题 → 核心数据 → 细节支撑，视觉引导线贯穿

**判断标准**：
> 如果去掉这个元素，用户获取信息会变慢吗？
> - 会 → 保留（这是有内容的密度）
> - 不会 → 删除（这是装饰）

## 四、细节签名（Signature Detail）

> **每个设计留一处「值得截图」的质感细节。**

这不是「精致到每个角落」，而是**在一个意想不到的地方做到极致**。

### 签名细节示例

| 物料类型 | 签名细节方向 | 具体做法 |
|---------|------------|---------|
| 品牌海报 | 排版上的意外 | 标题中的一个字用异体字/书法体打破统一 |
| 名片 | 触感暗示 | 极淡的纸张纹理背景 + 烫金/压凹的工艺标注 |
| 包装 | 开启仪式感 | 封口贴的设计、内衬的隐藏信息、撕拉条的巧妙处理 |
| 画册 | 页面节奏 | 每 8 页插入一页全出血大图，打破阅读节奏 |
| 宣传册 | 数据可视化 | 一个关键数字用超大字号 + 创意排版独占半页 |
| Logo | 负空间巧思 | 字母之间的负形构成一个隐藏图形 |

### 如何判断签名细节够好？

- 用户会主动说「这个细节很有意思」
- 截掉这个细节，设计整体仍完整；但加上它，设计有了灵魂
- 它不是你「加」上去的，而是从品牌故事中「长」出来的

## 五、Scale 规范

### 印刷物料

| 参数 | 最小值 | 理想值 | 说明 |
|------|--------|--------|------|
| 正文 | 8pt | 10-12pt | 中文最小 9pt，保证清晰 |
| 标题 | 18pt | 24-36pt | 视物料尺寸调整 |
| 行距 | 1.3倍 | 1.4-1.6倍 | 中文 1.4-1.6，英文 1.3-1.5 |
| 段间距 | 1.5倍字号 | 2倍字号 | 明显大于行距 |
| 页边距 | 10mm | 15-20mm | 不要太贴边 |
| 细线 | 0.25pt | 0.5pt | 印刷太细会断 |

### 数字物料（社交媒体/网页配图）

| 参数 | 最小值 | 理想值 | 说明 |
|------|--------|--------|------|
| 正文 | 12px | 14-16px | 老年人友好用 16px |
| 标题 | 24px | 36-48px | 视画布尺寸 |
| 行高 | 1.5 | 1.6-1.8 | 中文 1.7-1.8 |
| 留白 | 16px | 24-32px | 不要贴边 |
| 对比度 | 4.5:1 | 7:1 | WCAG AA 标准 |

### 海报/户外广告

| 参数 | 最小值 | 理想值 | 说明 |
|------|--------|--------|------|
| 主标题 | 视距 5m 可读 | 视距 10m 可读 | 按实际观看距离反推字号 |
| 副标题 | 视距 3m 可读 | 视距 5m 可读 | |
| 正文 | 视距 1m 可读 | 视距 2m 可读 | 海报正文通常很少 |
| 安全边距 | 画面 5% | 画面 10% | 防裁切 |

## 六、质感营造

### 印刷质感

| 质感方向 | 实现方式 | 适用 |
|---------|---------|------|
| 高级感 | 大面积留白 + 1 个 accent 色 + 衬线标题 | 奢侈品、画廊、高端服务 |
| 手工感 | 不规则边缘 + 手写元素 + 暖色调 | 烘焙、咖啡、文创、手作 |
| 科技感 | 网格系统 + 无衬线 + 冷色调 + 精确对齐 | 科技、金融、数据 |
| 自然感 | 有机形状 + 大地色系 + 宽松排版 | 有机、健康、户外 |
| 复古感 | 做旧纹理 + 复古字体 + 暖色调 + 边框装饰 | 酒吧、咖啡馆、复古品牌 |

### 数字质感

- **纸张纹理**：opacity ≤ 0.05 的 subtle 纹理增加温度
- **阴影层次**：`0 1px 2px rgba(0,0,0,0.04)`（卡片）→ `0 10px 40px rgba(0,0,0,0.1)`（ elevated）
- **玻璃拟态**（克制使用）：`backdrop-filter: blur(20px) saturate(150%)`

## 七、决策框架：当你不确定时

```
这个设计决策的品位测试：

1. 它服务于品牌识别吗？
   → 是 → 继续
   → 否 → 删除

2. 它是 120% 细节还是 80% 基础？
   → 120% → 确保只有一个这样的细节
   → 80% → 确保它达到行业合格线

3. 去掉它，设计会变差吗？
   → 是 → 保留
   → 否 → 删除

4. 用户能在 3 秒内说出这个设计的风格关键词吗？
   → 是 → 通过
   → 否 → 风格不够清晰，回到 Phase 2 重新定位
```

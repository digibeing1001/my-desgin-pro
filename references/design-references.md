## 七、国际顶级设计公司方法论（网站提取）

### 7.1 Pentagram — 全球最大独立设计公司

**3大设计哲学：**
- **Past as Prologue** — 过去是序章，历史是创新的资源
- **Type Has Spirit** — 字体有灵魂，排版是设计的基础语言
- **Secret Life of Things** — 事物的隐秘生活，日常物件中蕴含设计机会

**学科×行业矩阵（16学科×17行业）：**
- 学科：品牌/数字/产品/空间/影像/排版/声音/插画/动画/数据/出版/展览/交互/包装/标识/策略
- 行业：科技/金融/医疗/教育/文化/零售/食品/时尚/媒体/政府/交通/体育/酒店/非营利/法律/房地产/能源

### 7.2 Wolff Olins — 品牌转型叙事

**6类服务标签组合模式：**
- Brand Strategy + Visual Identity
- Brand Purpose + Employee Experience
- Brand Portfolio + Architecture
- Brand Experience + Digital Product
- Brand Naming + Verbal Identity
- Brand Culture + Internal Launch

**Inflection Point叙事框架：** 在转折点为品牌定义新方向

### 7.3 Landor — 品牌标语策略

**5种标语命名模式：**
- 描述型（What we do）
- 愿景型（Where we're going）
- 价值型（Why we matter）
- 行动型（What we inspire）
- 个性型（Who we are）

### 7.4 Brand New — 品牌重设计评估框架

**Before/After对比法：**
- Reviewed级：完整设计评论（优点+缺点+建议）
- Noted级：简要记录设计变更
- Spotted级：仅展示新设计

### 7.5 Braaands.co — 品牌叙事模板

**4步叙事法：**
1. 起源（Origin） — 品牌从何而来
2. 理念（Philosophy） — 品牌信仰什么
3. 视觉（Visual） — 品牌看起来怎样
4. 承诺（Promise） — 品牌承诺什么

**6大风格方向：**
- Minimalist / Bold / Playful / Elegant / Industrial / Organic

## 十、可操作的设计自检清单（从PDF实战规范提炼）

### 标志自检
- [ ] 标志是否使用了核准的设计源图？
- [ ] 安全空间是否严格遵守？
- [ ] 印刷最小尺寸是否达标（一般≥8mm）？
- [ ] 网络最小尺寸是否达标（一般≥40px）？
- [ ] 是否避免了10项常见禁令？
- [ ] 颜色优先级是否正确？

### 色彩自检
- [ ] 4种色值（PANTONE/CMYK/RGB/HEX）是否同时标注？
- [ ] 品牌色占比是否符合60-30-10法则？
- [ ] 浅色/深色背景下色彩比例是否调整？
- [ ] 渐变色是否有规范定义？

### 字体自检
- [ ] 中西文字体是否匹配？
- [ ] 5级字体层级是否完整？
- [ ] 备选字体是否指定（用于系统限制场景）？

### 应用物料自检
- [ ] 是否覆盖3大基础场景（印刷/数字/空间）？
- [ ] PPT模板是否有≥3种风格？
- [ ] 联合品牌规范是否定义？
- [ ] 数字端规范（社交媒体头像/EDM/网站）是否包含？
